## 第一节 复合命题的概述

### 一、复合命题的特征、组成

**【课本原文】** "所谓复合命题，就是包含有其他命题成分的命题，或者说，就是以命题作为其组成成分的命题。其表现形式相当于语言学中所说的复句。"

**【课本原文】** "一个复合命题中包含的命题成分，称作该复合命题的**肢命题**。"（⚠️ 注意：是"肢"不是"支"）

- 肢命题可以是简单命题，也可以是复合命题；可以是肯定命题，也可以是否定命题
- "若肢命题为复合命题的复合命题，称之为**多重复合命题**。"
- 肢命题用 p、q、r 等小写字母表示，是逻辑变项

**【课本原文】** "复合命题的逻辑性质同构成该复合命题的连接词有关，而与构成该复合命题的简单命题内部结构无关。"

### 二、复合命题真假的判定与真值表

**【课本原文】** "复合命题的真假，惟一地取决于它所包含的各肢命题的真假组合。"

- n 个肢命题 → 2ⁿ 种真假组合
- **真值表**："表明复合命题真值情况的图表，称之为真值表。"

五种基本逻辑联结词：

| 名称 | 符号 | 典型连接词 | 逻辑含义 |
|------|------|-----------|----------|
| 合取（联言）| ∧ | "并且" | 各肢命题同时为真 |
| 析取（选言）| ∨ | "或者" | 至少一个肢命题为真 |
| 蕴涵（假言）| → | "如果…那么…" | 前件蕴涵后件 |
| 逆蕴涵 | ← | "只有…才…" | 后件蕴涵前件 |
| 否定 | ¬ | "并非" | 否定原命题 |

## 第二节 复合命题的基本形式及其逻辑性质

### 一、联言命题

**【课本原文】** "联言命题就是同时断定两种以上事物情况都存在的复合命题。"

- 组成联言命题的肢命题叫**联言肢**
- 典型连接词："并且"；自然语言中也用"不但…而且…""既…又…""虽然…但是…"
- 符号表示：**p ∧ q**（合取）

**逻辑性质（真值表）**：

| p | q | p∧q |
|---|---|-----|
| + | + | **+** |
| + | - | - |
| - | + | - |
| - | - | - |

> （表中"+"表示真，"-"表示假，以下同）

> 📌 **"一假即假，全真才真"**——"当一个联言命题为真时，就意味着它的每一个肢命题都真；也只有当其每一个肢命题都真时，该联言命题也才真。"

**司法应用要点（课本原文）**：
1. 若联言肢互相矛盾 → 必为永假命题（永假式：p ∧ ¬p）
2. 反驳一个联言命题，只需证明其中一个联言肢为假即可

---

### 二、选言命题

**【课本原文】** "选言命题就是断定几种事物情况中至少有一种事物情况存在的命题。"

- 组成选言命题的肢命题叫**选言肢**
- 典型连接词："或者"
- 符号表示：**p ∨ q**（析取）

> ⚠️ **【重要】本书对选言命题不予分类。**
>
> 课本原文："本书对选言命题不予分类。"原因：（1）仅从命题形式很难确定是相容还是不相容，需结合具体内容才能判定；（2）无论相容还是不相容，共同特点都是断定"至少有一种事物情况存在"——"只要有一个选言肢为真，该选言命题就真；只有当各个选言肢都假，该选言命题才假。"

**逻辑性质（真值表）**：

| p | q | p∨q |
|---|---|-----|
| + | + | **+** |
| + | - | **+** |
| - | + | **+** |
| - | - | - |

> 📌 **"一真即真，全假才假"**

- 若选言肢为一对矛盾命题 → 必为永真命题（永真式：p ∨ ¬p）
- 穷尽选言肢是保证选言命题真实可靠的关键（尤其在侦查工作中）

**审判工作应用（课本原文）**：若法律条款属选言命题形式，"只要案件事实符合其中一个选言肢规定的情况，便可视为符合该项法律条款的规定；若要确认案件事实不符合该项法律条款的规定，则必须证明案件事实与每一选言肢规定的情形都不符合。"

---

### 三、假言命题

**【课本原文】** "假言命题就是断定两种事物情况之间存在着某种条件制约关系的命题。"（也叫**条件命题**）

- 前一部分肢命题 = **前件**（p），后一部分肢命题 = **后件**（q）
- 区别假言命题类型的**唯一依据是连接词**，不取决于对前、后件具体内容的分析

#### 客观事物情况之间的条件制约关系

**【课本原文】**

1. **充分条件**："若A、B两种事物情况之间，A情况出现或存在时，B情况就必然伴随着出现或存在，亦即'有A就必然有B'，则A情况就是制约B情况出现或存在的充分条件。"

2. **必要条件**："若A、B两种事物情况之间，A这一种事物情况不出现或不存在时，B这一种事物情况就必然不出现或不存在，亦即'无A便必然无B'，则A情况就是制约B情况出现或存在的必要条件。"

3. **充分条件与必要条件是相互逆反的关系**——当前件是后件的充分条件时，后件就是前件的必要条件；反之亦然。

#### （一）充分条件假言命题

**【课本原文】** "充分条件假言命题就是断定前件p是后件q的充分条件的假言命题。"

- 典型连接词："**如果……那么……**"；也用"只要……就……""若……则……"
- 符号表示：**p → q**（蕴涵）

**逻辑性质（真值表）**：

| p | q | p→q |
|---|---|-----|
| + | + | **+** |
| + | - | **-** |
| - | + | **+** |
| - | - | **+** |

> 📌 **"前件真而后件假"是唯一为假的情况**——"一个充分条件假言命题为假的惟一条件是'p'真并且'q'假。"
>
> "一个充分条件假言命题为真的条件是或者'p'假，或者'q'真。"

**反驳方法**：必须证明前件存在而后件却不存在。

#### （二）必要条件假言命题

**【课本原文】** "必要条件假言命题就是断定前件p是后件q的必要条件的假言命题。"

- 典型连接词："**只有……才……**"；也用"必须……才……"
- 符号表示：**p ← q**（逆蕴涵，读作"才"）

**逻辑性质（真值表）**：

| p | q | p←q |
|---|---|-----|
| + | + | **+** |
| + | - | **+** |
| - | + | **-** |
| - | - | **+** |

> 📌 **"前件假而后件真"是唯一为假的情况**——"必要条件假言命题为假的惟一条件是'p'假并且'q'真。"
>
> "必要条件假言命题为真的条件是或者'p'真，或者'q'假。"

**反驳方法**：必须证明前件不出现而后件却可以出现。

#### （三）关于充要条件假言命题

> ⚠️ **【重要】本书未列出充要条件假言命题。**
>
> 课本原文说明理由：（1）"当且仅当p，才q"在日常思维领域几乎难以得见；（2）实际应用中人们常从两方面同时断定（"如果p则q，并且只有p才q"），没有专用的日常连接词；（3）"从应用的角度来考虑就不仅不是必要的，而且容易导致人们去具体分析一个假言命题究竟是否充要条件的假言命题。"

## 第三节 复合命题的等值式及其应用意义

### 一、负命题及其等值式

**【课本原文】** "负命题就是否定某个命题的命题。"

- 符号表示：**¬p**（"并非p"）
- 典型连接词："并非"；也用"不可能""并不是""…是假的"
- "负命题与其肢命题的真假关系就是：若肢命题为真，则负命题为假；若肢命题为假，则负命题为真。"

**真值表**：

| p | ¬p |
|---|-----|
| + | - |
| - | + |

#### 复合命题的负命题等值式

| 原命题 | 负命题 | 等值于 |
|--------|--------|--------|
| p∧q | ¬(p∧q) | ¬p ∨ ¬q |
| p∨q | ¬(p∨q) | ¬p ∧ ¬q |
| p→q | ¬(p→q) | p ∧ ¬q |

> 📌 **德·摩根律**：
> - ¬(p∧q) ≡ ¬p ∨ ¬q
> - ¬(p∨q) ≡ ¬p ∧ ¬q

### 二、复合命题形式的转换及其应用意义

**【课本原文】** "所谓复合命题形式之间的转换，就是在不改变一个复合命题真假值的条件下，将其转换为另一个命题形式不同的复合命题。"

**司法工作意义（课本原文三点）**：
1. 有助于更好地理解一个复合命题所表示的断定，"在更深层次上把握一个复合命题实际断定的意思"
2. 有助于在同一思维过程中做到前后断定一致
3. 帮助从不同角度分析问题

### 三、多重复合命题及其等值式

**【课本原文】** "所谓多重复合命题，就是指由复合命题作肢命题而构成的复合命题。"

- 其负命题的等值式遵循基本复合命题负命题的等值规律

## 第四节 真值表方法 ⭐⭐

**真值表做题步骤（课本原文）**：
1. 确定命题变项（若有负命题，应设立负命题的肢命题）
2. 列出所有真假组合（n个变项 → 2ⁿ行）
3. 根据复合命题的构成顺序，由简到繁逐项列出
4. 依据肢命题真假组合和基本复合命题逻辑性质，确定各列真值
5. 根据最后一列判断命题性质

**三种判断结果**：

| 最后一列全部为… | 命题类型 | 含义 |
|-----------------|----------|------|
| 真 | **永真式**（重言式）| 无条件为真 |
| 假 | **永假式**（矛盾式）| 无条件为假 |
| 有真有假 | **可真可假式** | 取决于肢命题的真值 |

## 🧠 记忆诀窍

**联结词"一句话记忆"**：

| 联结词 | 课本术语 | 记法 |
|--------|----------|------|
| ∧ | 合取 | "全真才真" |
| ∨ | 析取 | "一真就真" |
| → | 蕴涵 | "前真后假才为假" |
| ← | 逆蕴涵（才）| "前假后真才为假" |
| ¬ | 否定 | "真假颠倒" |

**德·摩根律口诀**："横杠掰开，符号翻转"（¬ 分配进去，∧变∨，∨变∧）

**充分 vs 必要的区分诀窍**：
> "如果"后面是**充分**条件，"只有"后面是**必要**条件
> 充分条件假言命题 = "如果p，那么q" → p→q（唯一假：p真q假）
> 必要条件假言命题 = "只有p，才q" → p←q（唯一假：p假q真）

**永真式/永假式**：
> 选言命题选言肢为矛盾关系 → 永真式：p ∨ ¬p
> 联言命题联言肢为矛盾关系 → 永假式：p ∧ ¬p
