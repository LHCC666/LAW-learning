# 第一章 刑法概说

## 一、刑法的概念与性质

**定义**：规定犯罪、刑事责任和刑罚的法律规范的总称。

**三性质：**

| 性质 | 含义 |
|------|------|
| **公法** | 刑法调整国家与个人关系（与民法截然相反！） |
| **强行法** | 不允许当事人选择适用 |
| **保障法/后盾法** | 刑法是其他部门法的"最后一道防线" |

## 二、刑法渊源

1. **刑法典**（1997刑法 + 12个修正案）
2. **单行刑法**（如《关于惩治骗购外汇、逃汇和非法买卖外汇犯罪的决定》）
3. **附属刑法**（其他法律中的刑事条款）
4. **立法解释 + 司法解释**

## 三、刑法的体系

- 总则（第1-101条）+ 分则（第102-452条）+ 附则
- 法律条文结构：条→款→项

## 四、刑法的解释 ⭐

| 解释方法 | 定义 |
|----------|------|
| **文义解释** | 按条文通常字面含义解释 |
| **体系解释** | 联系刑法其他条文综合理解 |
| **历史解释** | 考察立法背景和沿革 |
| **目的解释** | 根据刑法保护的法益目的解释 |

**解释分类：**

| 分类标准 | 类型 |
|---------|------|
| 按解释主体 | 立法解释、司法解释、学理解释 |
| 按解释方法 | 文义解释、体系解释、历史解释、目的解释 |
| 按解释效果 | 扩张解释、限制解释、当然解释 |

## 五、民法与刑法的对比

|  | 民法 | 刑法 |
|--|------|------|
| 性质 | **私法** | **公法** |
| 调整关系 | 平等主体之间 | 国家与个人之间 |
| 核心功能 | 确权+救济 | 惩罚+预防 |
| 主体地位 | 平等 | 不平等（国家追诉） |

# 第二章 刑法基本原则

## 第一节 刑法基本原则概述

**概念**：贯穿全部刑法规范，指导和制约刑法适用的**根本准则**。

**法定依据**：《刑法》第3条—第5条明文规定。

## 第二节 保障人权原则

**核心含义**：刑法既要惩罚犯罪，也要**保障人权**——保障无罪的人不受刑事追究。

**三项具体体现：**
1. 罪刑法定原则的确立
2. 死刑适用的严格限制
3. 刑罚人道主义

> 这一原则是中国刑法学界的特色概括，体现"惩罚犯罪 + 保障人权"的双重使命。

## 第三节 罪刑法定原则 ⭐

《刑法》第3条："法律明文规定为犯罪行为的，依照法律定罪处刑；法律没有明文规定的，不得定罪处刑。"

**经典表述**：**法无明文规定不为罪，法无明文规定不处罚**（*Nullum crimen sine lege, nulla poena sine lege*）

**历史渊源**：1215年英国《大宪章》第39条 → 1810年《法国刑法典》第4条首次明文规定

### 六大派生原则

| 类别 | 派生原则 | 核心内容 |
|------|----------|----------|
| 传统 | ①成文法主义 | 犯罪及后果须由成文法律规定，排斥习惯法、判例法 |
| 传统 | ②禁止重法溯及既往 | 重法不适用于施行前行为（轻法可溯及既往） |
| 传统 | ③禁止有罪类推 | 禁止不利于被告人的类推（有利于被告的类推允许） |
| 传统 | ④禁止绝对不确定刑 | 要求相对确定刑 |
| 新发展 | ⑤明确性原则 | 刑法内容必须明确具体，不得含糊笼统 |
| 新发展 | ⑥适当性原则 | 禁止处罚不当罚行为 + 刑罚与犯罪相适应 |

### 司法适用要点

1. 司法人员不得超出刑法文本可能具有的含义范围适用刑法
2. **扩张解释 vs 类推适用**：一切超出一般公民预测可能性的解释 = 类推适用（被禁止），不等于合法的扩张解释
3. 司法解释必须具有明确性，不得以解释之名行类推之实

## 第四节 适用刑法人人平等原则

《刑法》第4条："对任何人犯罪，在适用法律上一律平等。"

| 层面 | 含义 |
|------|------|
| 定罪平等 | 同样行为同样定罪，不因身份而异 |
| 量刑平等 | 同样情节同样处罚 |
| 行刑平等 | 同样改造表现同样对待 |

## 第五节 罪责刑相适应原则 ⭐

《刑法》第5条："刑罚的轻重，应当与犯罪分子**所犯罪行**和**承担的刑事责任**相适应。"

| 环节 | 要求 |
|------|------|
| 立法体现 | 轻重有别的刑罚体系 + 区别对待的处罚制度 + 不同幅度法定刑 |
| 司法适用 | 综合分析社会危害性 + 犯罪人主观恶性 + 人身危险性 |
| 核心公式 | **重罪重罚、轻罪轻罚、罚当其罪** |

> 罪责刑相适应 = 报应（罪行） + 预防（刑事责任/人身危险性）的平衡

## 第六节 刑法的其他基本原则

### 一、罪责自负原则

刑罚只处罚犯罪者本人，不株连无辜。犯罪者承担的刑事责任是其个人责任。

### 二、主客观相统一原则

认定犯罪必须同时具备客观危害行为与主观罪过，二者缺一不可。反对客观归罪和主观归罪。

# 第三章 刑法效力范围

## 第一节 刑法的空间效力

### 一、空间效力的概念和原则

刑法空间效力解决刑法在什么地域、对什么人适用的问题。

**五大管辖原则：**

| 原则 | 依据 | 核心内容 |
|------|------|----------|
| **属地原则** | 领土主权 | 凡在中华人民共和国领域内犯罪的，适用我国刑法 |
| **属人原则** | 国籍 | 我国公民在我国领域外犯罪的，适用我国刑法 |
| **保护原则** | 保护国家/公民利益 | 外国人在我国领域外对我国国家或公民犯罪的，可适用我国刑法 |
| **普遍管辖原则** | 国际义务 | 对我国缔结或参加的国际条约所规定的罪行行使管辖权 |
| **折中原则** | 综合 | 以属地原则为主，兼采其他原则（我国采用） |

### 二、我国刑法的属地管辖权

- "领域"包括：领陆、领水、领空 + 船舶和航空器（旗国主义）
- "法律有特别规定"：享有外交特权和豁免权的外国人的刑事责任通过外交途径解决
- 犯罪地的认定：行为地或结果地有一项在我国领域内即可

### 三、我国刑法的属人管辖权

我国公民在我国领域外犯本法规定之罪的，适用本法，但按本法规定的最高刑为三年以下有期徒刑的，可以不予追究。

### 四、我国刑法的保护管辖权

外国人在我国领域外对我国国家或公民犯罪，而按本法规定的最低刑为三年以上有期徒刑的，可以适用本法，但是按照犯罪地的法律不受处罚的除外。

### 五、我国刑法的普遍管辖权

对我国缔结或参加的国际条约所规定的罪行，我国在所承担条约义务的范围内行使刑事管辖权。

## 第二节 刑法的时间效力

### 一、刑法的生效与失效

| 类型 | 说明 |
|------|------|
| 生效方式 | 自公布之日起生效 / 公布后过一段时间才生效 |
| 失效方式 | 明示废止 / 默示废止（新法取代旧法） |

### 二、刑法的溯及力 ⭐

**概念**：刑法生效后，对其生效前未经审判或判决未确定的行为是否适用。

**原则**：
- 从旧原则（行为时法）
- 从新原则（审判时法）
- 从旧兼从轻原则（**我国采用**）——原则上适用行为时的法律，但新法不认为是犯罪或处罚较轻的，适用新法

# 第四章 犯罪概念与犯罪构成

## 第一节 犯罪概念

### 一、犯罪概念的界定模式

| 模式 | 定义 | 代表 |
|------|------|------|
| **形式概念** | 犯罪是违反刑法、应受刑罚处罚的行为 | 大陆法系传统 |
| **实质概念** | 犯罪是具有社会危害性的行为 | 苏联法学 |
| **混合概念** | 形式+实质统一 | **我国采用** |

### 二、犯罪的基本特征

| 特征 | 内容 |
|------|------|
| **严重社会危害性** | 犯罪的本质特征，行为对刑法所保护的社会关系造成或可能造成严重损害 |
| **刑事违法性** | 违反刑法规范——"法无明文规定不为罪" |
| **应受刑罚惩罚性** | 犯罪行为应当受到刑罚处罚 |

> 三个特征的关系：社会危害性是基础→刑事违法性是法律表现→应受刑罚惩罚性是法律后果

## 第二节 犯罪构成

### 一、概念与特征

**概念**：依照我国刑法的规定，决定某一具体行为的社会危害性及其程度而为该行为构成犯罪所必需的一切客观要件和主观要件的有机统一体。

**四个特征**：法定性、主客观统一性、整体性、实质性与形式性统一

### 二、犯罪构成的四要件体系 ⭐

```
          犯罪构成
            │
  ┌────┬────┼────┬────┐
  │    │    │    │    │
犯罪  犯罪  犯罪  犯罪
客体  客观  主体  主观
      方面        方面
```

| 要件 | 核心内容 | 解决的问题 |
|------|---------|-----------|
| **犯罪客体** | 刑法所保护而为犯罪所侵犯的社会关系 | 侵犯了什么？ |
| **犯罪客观方面** | 危害行为、危害结果、因果关系等 | 怎样侵犯的？ |
| **犯罪主体** | 达到刑事责任年龄、具有刑事责任能力的人 | 谁侵犯的？ |
| **犯罪主观方面** | 故意、过失等心理态度 | 出于什么心态？ |

### 三、犯罪构成的分类

| 分类维度 | 类型 |
|---------|------|
| 按构成要素的单纯/混合 | 单纯的犯罪构成 vs 混合的犯罪构成 |
| 按是否完整 | 标准的犯罪构成 vs 修正的犯罪构成 |
| 按危害程度 | 普通的犯罪构成 vs 加重的犯罪构成 vs 减轻的犯罪构成 |

# 第五章 犯罪客体

## 第一节 犯罪客体的概念与特征

### 一、概念

犯罪客体是指我国刑法所保护而为犯罪行为所侵犯的社会关系。

### 二、特征

1. 犯罪客体是社会主义社会关系
2. 犯罪客体是我国刑法保护的社会关系
3. 犯罪客体是犯罪行为侵犯的社会关系

### 三、犯罪客体与犯罪对象 ⭐

| 对比维度 | 犯罪客体 | 犯罪对象 |
|---------|---------|---------|
| **性质** | 社会关系（抽象） | 具体的人或物（具体） |
| **是否必备** | 一切犯罪必备 | 部分犯罪没有对象 |
| **是否受侵害** | 必然受到侵害 | 不一定受到侵害 |
| **决定因素** | 决定犯罪性质 | 一般不决定犯罪性质 |
| **在分类中的作用** | 是犯罪分类的根据 | 不是分类根据 |

> 例如：盗窃罪——客体是财产所有权（社会关系），对象是被盗的财物（具体物）

## 第二节 犯罪客体的分类

| 层次 | 名称 | 含义 |
|------|------|------|
| 第一层 | **一般客体** | 一切犯罪共同侵犯的客体——我国刑法所保护的社会主义社会关系整体 |
| 第二层 | **同类客体** | 某一类犯罪共同侵犯的客体——刑法分则各章的分类依据 |
| 第三层 | **直接客体** | 某一具体犯罪直接侵犯的客体——决定具体罪名 |

**直接客体的分类：**
- 简单客体 vs 复杂客体（侵犯一个 vs 多个社会关系）
- 主要客体 vs 次要客体

# 第六章 犯罪客观方面

## 第一节 犯罪客观方面概述

### 一、概念

犯罪客观方面是指刑法规定的、说明行为对刑法所保护的社会关系造成侵害的客观外在事实特征。

### 二、特征

1. 法定性——由刑法明文规定
2. 客观性——外在的、可感知的客观事实
3. 多样性——不同犯罪有不同的客观要件
4. 关键地位——在犯罪构成四个要件中居于关键地位

### 三、意义

- 有助于区分罪与非罪
- 有助于区分此罪与彼罪
- 有助于正确分析和认定犯罪的罪过
- 有助于正确量刑

## 第二节 危害行为 ⭐

### 一、概念

**危害行为**：表现人的意志或意识、客观上侵犯刑法所保护的社会关系的身体动静。

**三个特征：**
1. 客观上是人的身体动静
2. 主观上是表现人的意志或意识的行为
3. 具有刑法禁止性

### 二、危害行为的基本形式

| 形式 | 定义 | 违反的规范 |
|------|------|-----------|
| **作为** | 积极实施刑法禁止的行为 | 禁止性规范 |
| **不作为** | 有义务实施且能够实施某种积极行为而未实施 | 命令性规范 |

**不作为的成立条件：**
1. 行为人负有实施特定积极行为的**作为义务**
2. 行为人**能够履行**该义务
3. 行为人**未履行**该义务

**作为义务的来源：**
1. 法律明文规定的义务
2. 职务或业务上要求的义务
3. 法律行为引起的义务（如合同）
4. 由先行行为引起的义务（先行行为致危险状态→有义务防止危害发生）

### 三、危害行为的分类

- 实行行为 vs 非实行行为
- 实行行为：刑法分则规定的具体犯罪构成要件的行为

## 第三节 危害结果

### 一、概念

**广义**：犯罪行为对刑法所保护的社会关系所造成的损害——包括构成要件结果和非构成要件结果、物质性和非物质性结果、直接和间接结果等。

**狭义（构成要件结果）**：由犯罪的实行行为造成的、根据刑法分则的规定对成立犯罪或者犯罪既遂具有决定意义的危害结果——只存在于过失犯罪、间接故意犯罪和结果犯的既遂犯中。

**【课本立场】** 危害结果不是一切犯罪构成的必要要件。

### 二、危害结果的四种分类

**（一）构成要件结果与非构成要件结果**

| | 构成要件结果（定罪结果） | 非构成要件结果（量刑结果） |
|--|----------------------|------------------------|
| 意义 | 对定罪具有决定意义 | 对量刑具有评价意义 |

**（二）物质性危害结果与非物质性危害结果**

| | 物质性（有形） | 非物质性（无形） |
|--|-------------|---------------|
| 特征 | 具体、可见、可计量 | 抽象、不可直接测量 |
| 举例 | 死亡、伤害、财产损害 | 名誉受损、人格侮辱 |

**（三）直接结果与间接结果**

| | 直接结果 | 间接结果 |
|--|---------|---------|
| 定义 | 危害行为本身直接引起 | 由直接结果进一步引起 |
| 与构成要件关系 | 构成要件结果只能是直接结果 | 只能是量刑情节 |

**（四）基本结果与加重结果**

| | 基本结果 | 加重结果 |
|--|---------|---------|
| 定义 | 作为犯罪构成要件的结果 | 引起另一种严重危害结果 |
| 法律效果 | 基本构成的要件 | 结果加重犯（以刑法明文规定为限） |

## 第四节 刑法上的因果关系 ⭐

### 一、概念

刑法上的因果关系，是指人的危害行为合乎规律地引起某种危害结果的内在联系。

### 二、因果关系的六大特点

| 特点 | 核心内容 |
|------|---------|
| **客观性** | 不以人的意志为转移的客观存在 |
| **相对性** | 必须从普遍联系的链条中抽取危害行为与危害结果这对现象 |
| **时间顺序性** | 原因在先，结果在后，不可颠倒 |
| **条件性** | 因果关系是具体的、有条件的，不能脱离案件具体条件 |
| **复杂性** | 一因多果（一个行为引起多种结果）、一果多因（多个原因引起一个结果） |
| **必然性和偶然性** | 因果关系本身是必然性和偶然性的统一 |

### 三、因果关系的性质

1. 作为原因的危害行为必须具有引起危害结果发生的**现实可能性**——因果之间具有质的同一性
2. 作为原因的危害行为必须**合乎规律地**引起危害结果的发生

### 四、因果关系的特殊认定

| 类型 | 认定规则 |
|------|---------|
| **假定的因果关系** | 即使没有该行为也会产生同样结果→仍应肯定因果关系的存在 |
| **重叠的因果关系** | 两个以上独立行为单独不能导致结果，合并导致结果→均肯定因果关系 |
| **流行病学的因果关系** | 公害犯罪中采用必要条件+病因学旁证法 |
| **不作为犯的因果关系** | 原因力在于"应该阻止而没有阻止事物向危险方向发展" |
| **中断的因果关系** | 综合三因素：①实行行为导致结果可能性高低 + ②介入因素异常性大小 + ③介入因素对结果贡献大小 |

### 五、因果关系与刑事责任

存在因果关系 ≠ 必然承担刑事责任，还需具备主观罪过。
