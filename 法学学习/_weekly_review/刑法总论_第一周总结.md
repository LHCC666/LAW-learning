# 第一章 刑法概说

## 第一节 刑法的概念、性质与渊源

### 一、刑法的概念

**刑法**是规定 **犯罪**、**刑事责任** 和 **刑罚** 的法律规范的总称。

### 二、刑法的性质

| 性质 | 含义 | 与民法的对比 |
|------|------|------------|
| **公法** | 刑法调整国家与个人之间的关系 | 民法调整平等主体之间的关系（**私法**） |
| **强行法** | 刑法规范不允许当事人选择适用或排除 | 民法中有大量任意性规范 |
| **保障法（后盾法）** | 刑法是其他部门法的"最后一道防线" | 民法是第一线调整手段 |

> **记忆口诀**："**公强盾**"——公法、强行法、后盾法（保障法）

### 三、刑法的渊源

| 序号 | 渊源 | 说明 |
|------|------|------|
| 1 | **刑法典** | 1997年《中华人民共和国刑法》+ 12个修正案 |
| 2 | **单行刑法** | 如《关于惩治骗购外汇、逃汇和非法买卖外汇犯罪的决定》 |
| 3 | **附属刑法** | 其他非刑事法律中的刑事条款 |
| 4 | **立法解释 + 司法解释** | 立法机关和最高司法机关的解释 |

---

## 第二节 刑法的体系与解释

### 一、刑法的体系

刑法由 **总则**（第1-101条）+ **分则**（第102-452条）+ **附则** 组成。

法律条文结构：**条 → 款 → 项**

### 二、刑法的解释

| 解释方法 | 定义 | 举例 |
|----------|------|------|
| **文义解释** | 按条文通常字面含义解释 | "故意杀人"→按字面理解 |
| **体系解释** | 联系刑法其他条文综合理解 | 总则"故意"→分则各罪"故意"参照总则定义 |
| **历史解释** | 考察立法背景和沿革 | 查阅修正案的立法理由书 |
| **目的解释** | 根据刑法所保护的法益目的解释 | 为什么某种行为被规定为犯罪？ |

> **记忆口诀**："**文体历目**"——文义解释、体系解释、历史解释、目的解释

# 第二章 刑法基本原则

## 第一节 刑法基本原则概述

刑法基本原则是贯穿全部刑法规范，指导和制约刑法适用的 **根本准则**。

**法定依据**：《刑法》第3条—第5条明文规定。

---

## 第二节 罪刑法定原则

> **【法条】** 《刑法》第3条：法律明文规定为犯罪行为的，依照法律定罪处刑；法律没有明文规定的，不得定罪处刑。

**经典表述**：**法无明文规定不为罪，法无明文规定不处罚**（*Nullum crimen sine lege, nulla poena sine lege*）

**历史渊源**：1215年英国《大宪章》第39条 → 1810年《法国刑法典》第4条首次明文规定

### 六大派生原则

| 类别 | 派生原则 | 核心内容 |
|------|----------|----------|
| 传统 | **成文法主义** | 犯罪及法律后果须由成文法律规定，排斥习惯法、判例法 |
| 传统 | **禁止重法溯及既往** | 重法不适用于施行前行为（轻法可溯及既往） |
| 传统 | **禁止有罪类推** | 禁止不利于被告人的类推（有利于被告人的类推允许） |
| 传统 | **禁止绝对不确定刑** | 要求相对确定刑 |
| 新发展 | **明确性原则** | 刑法内容必须明确具体，不得含糊笼统 |
| 新发展 | **适当性原则** | 禁止处罚不当罚行为 + 刑罚与犯罪相适应 |

> **记忆口诀**："**成不溯类定明当**"——成文法主义、不溯及既往、不类推、不确定刑、明确性、适当性

### 司法适用要点

1. **司法人员**：不得超出刑法文本可能具有的含义范围适用刑法
2. **扩张解释 vs 类推适用**：一切超出一般公民预测可能性的解释 = 类推适用（被禁止），不等于合法的扩张解释
3. **司法解释**：必须具有明确性，不得以解释之名行类推之实——类推是"法官造法"而非"法官释法"

---

## 第三节 适用刑法人人平等原则

> **【法条】** 《刑法》第4条：对任何人犯罪，在适用法律上一律平等。

| 层面 | 含义 |
|------|------|
| **定罪平等** | 同样行为同样定罪，不因身份而异 |
| **量刑平等** | 同样情节同样处罚 |
| **行刑平等** | 同样改造表现同样对待 |

---

## 第四节 罪责刑相适应原则

> **【法条】** 《刑法》第5条：刑罚的轻重，应当与犯罪分子**所犯罪行**和**承担的刑事责任**相适应。

| 环节 | 要求 |
|------|------|
| **立法体现** | 轻重有别的刑罚体系 + 区别对待的处罚制度 + 不同幅度的法定刑 |
| **司法适用** | 综合分析：社会危害性 + 犯罪人主观恶性 + 人身危险性 |
| **核心标准** | **重罪重罚、轻罪轻罚、罚当其罪** |

> 罪责刑相适应 = 报应（罪行） + 预防（刑事责任/人身危险性）的平衡

---

## 第五节 刑法的其他基本原则

### 一、罪责自负原则

- 刑罚只适用于犯罪人 **本人**
- **反对株连**，不株连无辜

### 二、主客观相统一原则

- 认定犯罪须 **同时具备**：客观危害行为 + 主观罪过
- **反对客观归罪**（只看结果不看罪过）
- **反对主观归罪**（只看罪过不看行为）

# 第三章 刑法的效力范围

## 第一节 刑法的空间效力

### 一、空间效力的概念

刑法的空间效力，是指刑法在 **什么地域**、对 **什么人** 适用的问题。

### 二、四大管辖原则

| 原则 | 含义 | 法律依据 |
|------|------|---------|
| **属地原则** | 凡在中华人民共和国**领域内**犯罪的，适用我国刑法 | 《刑法》第6条 |
| **属人原则** | 中华人民共和国**公民**在领域外犯罪的，适用我国刑法 | 《刑法》第7条 |
| **保护原则** | 外国人在我国领域外对我国**国家或公民**犯罪的，适用我国刑法 | 《刑法》第8条 |
| **普遍管辖原则** | 对于我国缔结或参加的国际条约所规定的**国际犯罪**，我国行使刑事管辖权 | 《刑法》第9条 |

### 三、属地原则的细节

| 概念 | 含义 |
|------|------|
| **领域** | 领陆、领水、领空 + 我国船舶和航空器（**旗国主义**） |
| **犯罪地标准** | 行为或结果有一项发生在我国境内即可（**遍在主义**） |
| **例外** | 享有外交特权和豁免权的外国人的刑事责任通过外交途径解决（第11条） |

> **管辖优先级**：属地 > 属人 > 保护 > 普遍管辖

---

## 第二节 刑法的时间效力

### 一、刑法的生效与失效

- **生效**：公布之日起生效 或 公布后经过一定期限生效
- **失效**：明示废止 或 新法取代旧法（默示废止）

### 二、刑法的溯及力

**概念**：新的刑法制定后，对其生效前未经审判或判决尚未确定的行为是否适用。

### 三、从旧兼从轻原则

> **【法条】** 《刑法》第12条：中华人民共和国成立以后本法施行以前的行为，如果当时的法律不认为是犯罪的，适用当时的法律；如果当时的法律认为是犯罪的，依照本法总则第四章第八节的规定应当追诉的，按照当时的法律追究刑事责任，但是如果本法不认为是犯罪或者处刑较轻的，适用本法。

| 情形 | 适用规则 |
|------|---------|
| 旧法不认为是犯罪 | 适用 **旧法**（从旧） |
| 旧法认为是犯罪，新法不认为是犯罪 | 适用 **新法**（从轻） |
| 新旧法都认为是犯罪 | 原则上适用旧法（从旧），但新法处刑较轻的适用新法（**从轻**） |

# 第四章 犯罪概念与犯罪构成

## 第一节 犯罪的概念

> **【法条】** 《刑法》第13条：一切危害国家主权、领土完整和安全，分裂国家、颠覆人民民主专政的政权和推翻社会主义制度，破坏社会秩序和经济秩序，侵犯国有财产或者劳动群众集体所有的财产，侵犯公民私人所有的财产，侵犯公民的人身权利、民主权利和其他权利，以及其他危害社会的行为，依照法律应当受刑罚处罚的，都是犯罪，但是情节显著轻微危害不大的，不认为是犯罪。

### 犯罪的基本特征（三特征）

| 特征 | 含义 | 地位 |
|------|------|------|
| **严重的社会危害性** | 行为对刑法所保护的社会关系造成或可能造成严重损害 | 犯罪的 **本质特征**（最根本） |
| **刑事违法性** | 行为违反刑法规范，符合刑法规定的犯罪构成 | 犯罪的 **法律特征**（形式特征） |
| **应受刑罚处罚性** | 行为应当受到刑罚处罚 | 犯罪的 **法律后果特征** |

> **"但书"**：情节显著轻微危害不大的，不认为是犯罪。

---

## 第二节 犯罪构成

### 一、犯罪构成的概念

**犯罪构成**是刑法规定的，决定某一具体行为成立犯罪所必需的 **主观要件** 和 **客观要件** 的有机统一体。

### 二、犯罪构成与犯罪概念的关系

| 对比 | 犯罪概念 | 犯罪构成 |
|------|---------|---------|
| **层面** | 回答"什么是犯罪" | 回答"犯罪是怎样成立的" |
| **功能** | 揭示犯罪的本质特征 | 提供认定犯罪的具体规格和标准 |
| **关系** | 犯罪概念是犯罪构成的基础 | 犯罪构成是犯罪概念的具体化 |

### 三、犯罪构成的四要件体系

```
                      犯罪构成
                          |
        +---------+-------+-------+---------+
        |         |       |       |         |
    犯罪客体   犯罪客观   犯罪主体   犯罪主观
               方面                方面
```

| 要件 | 含义 | 核心内容 |
|------|------|---------|
| **犯罪客体** | 刑法所保护而为犯罪行为所侵害的社会关系 | 第五章 |
| **犯罪客观方面** | 刑法规定的、说明行为对刑法所保护的社会关系造成侵害的客观外在事实特征 | 危害行为、危害结果、因果关系（第六章） |
| **犯罪主体** | 实施危害行为，依法应当负刑事责任的自然人和单位 | 刑事责任年龄、刑事责任能力（第七章） |
| **犯罪主观方面** | 犯罪主体对其实施的危害行为及其危害结果所持的心理态度 | 故意、过失、目的、动机（第八章） |

### 四、犯罪构成要件的分类

| 分类 | 含义 |
|------|------|
| **基本要件 vs 修正要件** | 基本要件 = 单独犯的既遂状态；修正要件 = 共犯、未遂犯 |
| **积极的构成要件 vs 消极的构成要件** | 积极的 = 肯定犯罪成立；消极的 = 否定犯罪成立（如正当防卫） |
| **描述性要件 vs 规范性要件** | 描述性 = 事实判断（如"杀人"）；规范性 = 价值判断（如"淫秽物品"） |

# 第五章 犯罪客体

## 第一节 犯罪客体的概念

**犯罪客体**，是指我国刑法所保护的、为犯罪行为所侵害的 **社会关系**。

### 犯罪客体的特征

1. **犯罪客体是一种社会关系** — 是人们在社会生活中形成的相互关系
2. **犯罪客体是刑法所保护的社会关系** — 不是所有的社会关系都是犯罪客体
3. **犯罪客体是犯罪行为所侵害的社会关系** — 社会关系受到实际侵害或威胁时才成为犯罪客体

---

## 第二节 犯罪客体的分类

| 类型 | 含义 | 举例 |
|------|------|------|
| **一般客体** | 一切犯罪所共同侵害的社会关系整体 | 刑法所保护的社会关系的总和 |
| **同类客体** | 某一类犯罪所共同侵害的社会关系的某一部分 | 侵犯财产罪的同类客体 = 财产所有权 |
| **直接客体** | 某一具体犯罪所直接侵害的特定社会关系 | 盗窃罪的直接客体 = 公私财产所有权 |

> **三层次关系**：一般客体（宏观）→ 同类客体（中观，刑法分则分章标准）→ 直接客体（微观，个罪认定）

---

## 第三节 犯罪客体与犯罪对象的区别

| 对比维度 | 犯罪客体 | 犯罪对象 |
|---------|---------|---------|
| **本质** | 刑法保护的社会关系（**抽象**） | 犯罪行为直接作用的具体物或人（**具体**） |
| **是否必备** | **任何犯罪都必须有犯罪客体** | 不是所有犯罪都有犯罪对象（如脱逃罪） |
| **是否受侵害** | 任何犯罪都侵害犯罪客体 | 犯罪对象不一定受到侵害（如盗窃罪中物品完好） |
| **定罪作用** | 决定犯罪性质 | 一般不决定犯罪性质（同一对象可体现不同客体） |

> **典型例子**：盗窃仓库中的电线（破坏电力设备？还是盗窃？）
> - 仓库中未使用的电线 = 侵犯财产所有权（盗窃罪）
> - 正在使用的电线 = 侵犯公共安全（破坏电力设备罪）
> - 对象相同（电线），客体不同 → 罪名不同！

# 第六章 犯罪客观方面

## 第一节 犯罪客观方面概述

**犯罪客观方面**，是指刑法规定的、说明行为对刑法所保护的社会关系造成侵害的客观外在事实特征。

**地位**：犯罪客观方面是 **犯罪构成的核心**，是一切犯罪必备的共同要件。

### 犯罪客观方面的要件分类

| 类型 | 含义 | 举例 |
|------|------|------|
| **必备要件** | 一切犯罪都必须具备的 | 危害行为 |
| **选择要件** | 某些犯罪需要具备的 | 危害结果、因果关系、犯罪时间、地点、方法 |

---

## 第二节 危害行为

### 一、危害行为的概念

**危害行为**，是指行为人在自己意识、意志支配下实施的，客观上侵害或威胁刑法所保护的社会关系的 **身体动静**。

### 二、危害行为的特征

1. **有体性** — 表现为人的身体动静（客观外在）
2. **有意性** — 受行为人的意识和意志支配
3. **有害性** — 对刑法所保护的社会关系造成侵害或威胁

> **排除**：反射动作、睡梦中的行为、绝对强制下的行为、无意识的动作

### 三、危害行为的三种基本形式

| 形式 | 定义 | 分类 | 核心判断 |
|------|------|------|---------|
| **作为** | 行为人以积极的身体动作实施刑法禁止的行为 | 绝大多数犯罪 | 违反 **禁止性规范**（做了不该做的） |
| **不作为** | 行为人消极地不履行特定的法律义务 | 纯正不作为、不纯正不作为 | 违反 **命令性规范**（没做该做的） |
| **持有** | 行为人对特定物品事实上支配和控制的状态 | 如非法持有毒品罪 | 第三种独立形式 |

### 四、不作为犯的成立条件

1. 行为人负有实施特定积极行为的 **法律义务**
2. 行为人 **有能力履行** 该项义务
3. 行为人 **未履行** 该项义务
4. 因不履行义务而造成了或可能造成危害结果

> **不作为的义务来源**：① 法律明文规定 ② 职务或业务要求 ③ 合同行为 ④ 先行行为引起的义务

---

## 第三节 危害结果

### 一、危害结果的概念

**危害结果**，是指危害行为对刑法所保护的社会关系所造成的 **实际损害** 或 **现实威胁**。

### 二、危害结果的分类

| 分类标准 | 类型 | 含义 |
|---------|------|------|
| 以是否属于构成要件 | **构成结果** | 属于犯罪构成要件的结果 |
| | **非构成结果** | 不属于构成要件的结果 |
| 以存在形态 | **物质性结果** | 具体、可见、可计量的结果（如死亡、伤害） |
| | **非物质性结果** | 抽象、不可直接测量的结果（如名誉受损） |
| 以因果链条 | **直接结果** | 危害行为本身直接引起的结果 |
| | **间接结果** | 由直接结果进一步引起的结果 |
| 以作用性质 | **普通结果** | 作为犯罪构成要件的基本结果 |
| | **加重结果** | 引起另一种更严重危害结果（以刑法明文规定为限） |

---

## 第四节 刑法上的因果关系

### 一、因果关系的概念

**刑法上的因果关系**，是指危害行为与危害结果之间存在的 **引起与被引起** 的客观联系。

### 二、因果关系的六个特点

| 特点 | 含义 |
|------|------|
| **客观性** | 因果关系是客观存在的，不以人的意志为转移 |
| **相对性** | 从普遍联系中抽取危害行为与危害结果环节进行考察 |
| **顺序性** | 原因在前，结果在后（先因后果） |
| **多样性** | 一因一果、一因多果、多因一果、多因多果 |
| **复杂性** | 存在偶然因果、间接因果等复杂形态 |
| **必然性与偶然性统一** | 存在合乎规律的内在联系 |

### 三、特殊因果关系的认定

| 类型 | 含义 | 处理 |
|------|------|------|
| **中断的因果关系** | 因果关系进程中介入其他原因，切断原来因果进程 | 综合三因素判断：① 实行行为导致结果的可能性高低 ② 介入因素异常性大小 ③ 介入因素对结果的贡献大小 |
| **偶然因果关系** | 危害行为偶然地与其他因素结合而导致危害结果 | 一般不影响定罪，但影响量刑 |
| **不作为的因果关系** | 行为人不履行特定义务而导致危害结果 | 须以行为人负有作为义务为前提 |

### 四、因果关系与刑事责任的关系

- 因果关系是客观基础 — 解决的是 **客观归责** 问题
- 刑事责任还包括主观要件 — 还需要 **故意或过失**
- **不能以因果关系代替主观罪过**，也不能以主观罪过代替因果关系
