# 第一章 民法概述

## 一、民法的概念

- **形式民法**：以"民法"命名的法典（《民法典》）
- **实质民法**：所有调整民事关系的法律规范总称（《民法典》+ 民事单行法 + 司法解释）
- 我国《民法典》2020年5月28日通过，2021年1月1日施行，7编1260条

## 二、调整对象

民法调整**平等主体之间**的人身关系和财产关系。
- 人身关系：人格权、身份权
- 财产关系：物权、债权、继承权、知识产权
- 不包括：行政隶属关系、财政税收关系（纵向不平等关系）

## 三、民法的性质

| 性质 | 含义 |
|------|------|
| **私法** | 调整平等主体关系，区别于调整国家公权力的公法 |
| **权利法** | 以确认和保护民事权利为核心（民法典是"权利宣言书"） |
| **市场经济基本法** | 确认市场主体的法律地位，规范交易行为 |

## 四、民法典的体系

- **潘德克顿体系**（德国式，我国采用）：总则→分则，提取公因式
  - 总则编 + 物权编 + 合同编 + 人格权编 + 婚姻家庭编 + 继承编 + 侵权责任编
- **法学阶梯体系**（罗马式/法国式）：人→物→诉讼
- 我国特色亮点：人格权独立成编（全球首创）、侵权责任独立成编

## 五、民法渊源

1. 法律（《民法典》+ 民事单行法）
2. 行政法规
3. 司法解释（最高人民法院）
4. 习惯（《民法典》第10条："法律没有规定的，可以适用习惯，但是不得违背公序良俗"）

## 六、民法与刑法的对比

|  | 民法 | 刑法 |
|--|------|------|
| 性质 | **私法** | **公法** |
| 调整关系 | 平等主体之间 | 国家与个人之间 |
| 核心功能 | 确权+救济 | 惩罚+预防 |

# 第二章 民法基本原则

## 第一节 民法基本原则概述

基本原则的性质：**强行性规范**，不允许当事人排除适用。

**司法适用规则：**
1. 有具体规则必须适用具体规则，不能直接援引基本原则裁判
2. 基本原则可与具体规则结合适用（作为说理依据）
3. 法律规定不明确时，原则为解释提供价值指引
4. 存在法律漏洞时，原则可作为填补漏洞的兜底方法

## 第二节 民事权益受法律保护原则

《民法总则》第3条，置于基本原则之首。

**四层内涵：**
1. 保护人身、财产等权益——民法主要保护私权
2. 不仅保护权利，也保护利益——"其他合法权益"保持开放性
3. 对新型民事权益的保护——隐私权、个人信息保护
4. 通过民事责任进行救济——侵害→民事责任→保护

## 第三节 平等原则

《民法总则》第4条："民事主体在民事活动中的法律地位一律平等。"

| 层面 | 内容 |
|------|------|
| 人格平等 | 自然人民事权利能力一律平等（不分尊卑贵贱） |
| 法律关系地位 | 具体法律关系中当事人地位平等，不得强加意志 |
| 平等对待 | 强势平等（不分类）+ 弱势平等（区别对待以实现实质平等） |
| 补救平等 | 权利受侵害时一体保护，不因所有制而异 |

## 第四节 意思自治原则 ⭐

《民法总则》第5条。也称"私法自治原则"。

**核心含义**：民事主体在法定范围内有广泛行为自由，可依自己意志产生、变更、消灭民事法律关系。

**三个内容维度：**
1. 赋予广泛行为自由——有权从事/不从事民事活动，有权选择行为内容和相对人，有权选择行为方式，有权选择补救方式
2. 通过法律行为调整关系——任意性调整方法，只划定界限
3. 划定行政干预的界限——**法无明文禁止即为自由**

意思自治 vs 私法自治：意思自治是私法自治的组成部分，但不等同。自20世纪以来，意思自治受到国家干预的限制。

## 第五节 公平原则

《民法总则》第6条。

| 领域 | 具体体现 |
|------|----------|
| 合同法 | 等价有偿、情事变更、显失公平制度 |
| 物权法 | 添附中的补偿、相邻关系中的容忍义务 |
| 侵权法 | 公平责任（双方无过错时）、完全赔偿原则、损益相抵 |

公平原则更多体现**实质正义**的要求。

## 第六节 诚实信用原则 ⭐

《民法总则》第7条。**"帝王规则"**，君临法域。

**两层核心要求**：秉持诚实 + 恪守承诺

**四大功能：**

| 功能 | 说明 |
|------|------|
| ①确立行为规则 | 产生附随义务（先契约→履约→后契约）、物权行使规则、安全保障义务 |
| ②填补法律和合同漏洞 | 内容遗漏/约定不明确时可依诚信原则填补 |
| ③衡平功能 | 平衡当事人之间+当事人与社会之间的利益冲突 |
| ④解释功能 | 解释法律和合同的准则 |

诚信原则是**强行性规范**，当事人不得以协议排除或规避。

## 第七节 符合法律和公序良俗原则

《民法总则》第8条 + 第153条第2款（违背公序良俗的民事法律行为无效）。

**公序良俗 = 公共秩序 + 善良风俗**

| 维度 | 诚信原则 | 公序良俗原则 |
|------|----------|------------|
| 适用范围 | 更宽（民法各领域） | 主要在人身关系和债法 |
| 功能 | 填补法律/合同漏洞，产生新规则 | 主要认定法律行为效力 |
| 调整方式 | 设定积极行为义务 | 禁止从事某种行为 |
| 保护目的 | 保护对方当事人利益 | 保护第三人和公共利益 |

## 第八节 绿色原则

《民法总则》第9条："民事主体从事民事活动，应当有利于节约资源、保护生态环境。"

**两层内涵**：有效率地利用资源 + 保护环境和生态

> 这是中国民法典的时代特色，回应21世纪生态危机，传承"天地人和"传统理念。

# 第三章 民事法律关系

## 第一节 民事法律关系的一般原理

**概念**：民事法律关系是由民法规范调整的社会关系，也就是由民法确认和保护的社会关系。

**四个特征：**
1. 是民法所调整的社会关系在法律上的表现
2. 是人与人之间的权利义务关系（拉伦茨："人与人之间的法律纽带"）
3. 具有平等性
4. 具有一定程度的任意性（发生/变更/消灭/内容四个方面的任意性）

**在民法体系构建中的意义：**
1. 是整个民法逻辑体系展开与构建的基础（萨维尼最早系统提出）
2. 对整个民法学的建立和发展具有重要意义
3. 是指导理论研究人员与司法实务工作者解决实践问题的基本思维模式

## 第二节 民事法律关系的要素

**【课本立场】** 三要素说：主体 + 客体 + 内容

| 学说 | 内容 | 课本评价 |
|------|------|----------|
| 三要素说 | 主体 + 客体 + 内容 | ✅ 课本采纳 |
| 四要素说 | + 责任 | 责任是"原有的法律关系的变异形态" |
| 五要素说 | + 变动 + 变动原因 | 法律事实"是外在于法律关系的" |

**（一）主体**：参加民事法律关系，享有民事权利并承担民事义务的人。包括公民、法人和非法人组织，国家在一些场合也是特殊主体。

**（二）内容**：民事主体所享有的权利和承担的义务。《民法总则》第131条："民事主体行使权利时，应当履行法律规定的和当事人约定的义务。"

**（三）客体**：民事权利和民事义务所指向的对象。区分不同法律关系确定：

| 法律关系类型 | 客体 |
|-------------|------|
| 物权法律关系 | 物 |
| 知识产权法律关系 | 智力成果 |
| 债权法律关系 | 体现一定物质利益的行为 |

## 第三节 民事法律事实

**概念**：依法能够引起民事法律关系产生、变更或消灭的客观现象。

**三个特点**：客观存在（非主观内心意思）、能够引起法律后果、取决于法律规定。

**分类：**

```
民事法律事实
├── 事件（自然事实）：与人的意志无关
│   例：人的死亡→继承；物的灭失→所有权消灭
└── 行为：人的有意识的活动
    ├── 民事法律行为（以意思表示为要素）
    ├── 准法律行为（意思通知、观念通知、感情表示）
    └── 事实行为（不以意思表示为要素）
```

# 第五章 自然人

## 第一节 自然人的民事权利能力

### 一、自然人的概念

依自然规律产生，具有五官百骸，区别于其他动物的人。既是生物学概念，又是法律概念。"自然人"的概念优于"公民"：更符合民法的私法特点，更有利于彰显民事权利能力平等的理念。

### 二、自然人民事权利能力的概念和特征

**【课本原文】** 自然人依法享有民事权利和承担民事义务的资格，是每个自然人平等地享有民事权利、承担民事义务的可能性。

| 特征 | 核心含义 |
|------|---------|
| 平等性 | 不分性别、民族、出身、职业、财产状况，法律地位一律平等（第14条） |
| 普遍性 | 外国人和无国籍人也具有民事权利能力 |
| 不可剥夺性 | 刑事犯罪可限制自由，但不能剥夺民事权利能力 |
| 不得转让和抛弃 | 权利能力与主体资格不可分离 |
| 内容广泛性 | 可依法享有各种人身权利和财产权利 |

### 三、民事权利能力与权利的区别

| 比较维度 | 民事权利能力 | 民事权利 |
|---------|-------------|---------|
| 性质 | 享有权利的资格/可能性 | 已实际享有的现实权利 |
| 内容 | 包括享有权利+承担义务的能力 | 仅涉及享有的权利 |
| 来源 | 国家通过法律直接赋予 | 参与具体法律关系才能享有 |
| 存续期间 | 始于出生、终于死亡 | 由特定法律事实决定 |
| 可处分性 | 不可转让、不得放弃 | 除另有规定外，可自行处分 |

### 四、权利能力的开始与终止

- 始于出生（《民法总则》第13条），终于死亡
- **胎儿利益保护**（第16条）：涉及遗产继承、接受赠与等胎儿利益时，胎儿视为具有民事权利能力；娩出时为死体的，自始不存在
- 死亡时间的认定：以死亡证明记载的时间为准

## 第二节 自然人的民事行为能力

### 一、概念和特点

民事行为能力是自然人**以自己的行为**独立行使民事权利和承担民事义务的资格。

### 二、三档划分

| 类型 | 条件 | 行为效力 |
|------|------|----------|
| **完全民事行为能力** | 18周岁以上；16周岁以上以自己劳动收入为主要生活来源的视为完全 | 可独立实施民事法律行为 |
| **限制民事行为能力** | 8周岁以上未成年人；不能完全辨认自己行为的成年人 | 纯获利益+与其年龄智力相适应的行为有效；其他需法定代理人同意或追认 |
| **无民事行为能力** | 不满8周岁；不能辨认自己行为的成年人 | 民事法律行为由法定代理人代理实施 |

### 三、民事行为能力的宣告与恢复

- 可申请认定不能辨认/不能完全辨认自己行为的成年人为无/限制民事行为能力人
- 恢复后可申请恢复民事行为能力

## 第三节 监护

### 一、概念与特征

监护是为无民事行为能力人和限制民事行为能力人设立保护人的制度。

### 二、监护的分类

| 类型 | 适用对象 |
|------|---------|
| 未成年人监护 | 父母是当然监护人；父母死亡/无监护能力的依次由祖父母、兄姐等担任 |
| 成年监护 | 无/限制民事行为能力的成年人 |

### 三、监护的设定方式

| 方式 | 说明 |
|------|------|
| **法定监护** | 法律直接规定监护人范围和顺序 |
| **意定监护** | 具有完全民事行为能力的成年人，可事先协商以书面形式确定自己的监护人 |
| **遗嘱监护** | 父母可通过遗嘱指定监护人 |
| **指定监护** | 对监护人有争议时，由居委会/村委会/民政部门/法院指定 |

### 四、监护人职责

- 代理被监护人实施民事法律行为
- 保护被监护人的人身权利、财产权利及其他合法权益
- 除为维护被监护人利益外，不得处分被监护人的财产
- 监护资格可撤销（严重损害被监护人权益等情形）

## 第四节 宣告失踪与宣告死亡

| 对比维度 | 宣告失踪 | 宣告死亡 |
|---------|---------|---------|
| **条件** | 下落不明满2年 | 下落不明满4年（意外事故满2年，经证明不可能生存的不受时间限制） |
| **申请人** | 利害关系人 | 利害关系人 |
| **法律效果** | 指定财产代管人，管理失踪人财产 | 产生与自然死亡相同的法律效果（婚姻关系消灭、遗产继承开始等） |
| **撤销效果** | 财产代管关系终止 | 婚姻关系视情况恢复；已继承财产应返还 |

## 第五节 个体工商户和农村承包经营户

- 个体工商户：自然人在法律允许范围内依法经核准登记从事工商业经营
- 农村承包经营户：农村集体经济组织成员依法承包经营集体所有或国家所有的土地
- 债务承担：个人经营的以个人财产承担；家庭经营的以家庭财产承担

## 第六节 自然人的住所

- 住所：自然人生活和进行民事活动的主要基地和中心场所
- 经常居所与住所不一致的，经常居所视为住所
- 无经常居所的，以户籍登记或其他有效身份登记记载的居所为住所

# 第十一章 民事权利

## 第一节 民事权利概述

**概念**：由国家强制力予以保障的类型化的民事主体所享有的利益。

**五个特征：**
1. 由民法所确认
2. 民事主体所享有的利益
3. 体现为一定范围内的行为自由
4. 受到国家强制力的保障
5. 是类型化了的利益

## 第二节 民事权利的基本分类

### 一、财产权、人身权和综合性权利

### 二、支配权、请求权、抗辩权、形成权 ⭐

| 权利类型 | 定义 | 特点 | 举例 |
|---------|------|------|------|
| **支配权** | 直接支配权利客体、排除他人干涉 | 绝对性、排他性 | 物权、人格权、知识产权 |
| **请求权** | 请求他人为或不为一定行为 | 相对性、需他人配合 | 债权请求权、物权请求权 |
| **抗辩权** | 对抗请求权、阻止其效力 | 防御性、须权利人主张 | 诉讼时效抗辩、同时履行抗辩 |
| **形成权** | 单方意思表示即可变动法律关系 | 单方性、不可撤销 | 撤销权、追认权、解除权、抵销权 |

### 三、绝对权与相对权

| | 绝对权 | 相对权 |
|--|--------|--------|
| 义务人 | 不特定的任何人 | 特定的人 |
| 内容 | 要求他人不得干涉 | 要求特定人为一定行为 |
| 举例 | 物权、人格权 | 债权 |

### 四、主权利与从权利、既得权与期待权

## 第三节 民事权利的主要类型

### 一、人身权
- **人格权**：生命权、身体权、健康权、姓名权、肖像权、名誉权、荣誉权、隐私权、婚姻自主权等
- **身份权**：配偶权、亲权、亲属权等

### 二、物权
- 所有权、用益物权、担保物权
- 物权法定原则 + 物权平等保护原则

### 三、债权
- 合同之债、侵权之债、无因管理之债、不当得利之债

### 四、知识产权
- 著作权、专利权、商标权等

### 五、继承权及其他权益

## 第四节 民事权利的取得、变动与消灭

- **取得**：原始取得（直接依法律规定取得）与继受取得（从前手取得）
- **变更**：权利主体、客体或内容的变更
- **消灭**：绝对消灭（权利本身不存在）与相对消灭（权利转移给他人）

## 第五节 民事权利的行使与保护

**行使原则**：自愿原则、义务必须履行原则、禁止权利滥用原则

**保护方式**：自我保护（私力救济）+ 国家保护（公力救济，通过诉讼等方式）
