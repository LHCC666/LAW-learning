### 🔑 核心知识点精要

#### 第一节 法人的概念和性质

**一、法人的概念**

> **【课本原文·法条】** 《民法典》第57条：法人是具有民事权利能力和民事行为能力，依法独立享有民事权利和承担民事义务的组织。

**【课本原文】第57条揭示的四项基本特征：**

1. **独立的名义**——法人能够以自己的名义独立地享有权利、承担义务，并能在法院起诉应诉。法人具有组织上的统一性，从而极大地降低了交易费用。
2. **独立的财产**——法人所具有的独立于其投资人以及法人的成员的财产。法人独立的财产区别于法人成员的财产，法人财产的独立性越明确，就越能够防止法人成员及设立人侵害法人的财产。由于法人的财产与出资人的财产分开，在法人内部要实行所有权与经营权分离，建立健全的组织结构。
3. **健全的组织机构**——对内管理法人的事务、对外代表法人从事民事活动的机构的总称。营利法人的组织机构实行决策、执行、监督机构相互独立、权责分明、相互制衡的机制。
4. **独立的责任**——《民法典》第60条：法人以其全部财产独立承担民事责任。法人的成员及设立人仅以其出资或认缴的出资为限对法人债务承担责任。法人成员承担有限责任是法人的典型形式，也是区分法人和非法人组织的重要特征。

**二、法人的性质（三学说）**

| 学说 | 核心观点 | 对行为能力的看法 |
|------|---------|-----------------|
| **拟制说** | 法人是法律拟制的人格，无意思属性 | 法人本身无行为能力，须由自然人代理 |
| **否认说** | 法人无独立人格，只是"目的财产" | 实际权利义务归属于管理财产的自然人 |
| **实在说** | 法人是客观存在的主体，有团体意志 | 法人机构的行为就是法人自己的行为 |

> **【课本立场】** 我国采纳"实在说"，但例外情况下可以吸收拟制说的合理之处——法人机构违反法人意志损害法人利益的行为，不应一概由法人承受。

**三、法人的分支机构**

- 《民法典》第74条第1款：法人可以依法设立分支机构
- 区分：领取营业执照的 vs 未领取营业执照的（三点区别：能否独立从事民事活动/能否独立诉讼/责任承担顺序）
- 第74条第2款：分支机构以自己名义从事民事活动，产生的民事责任由法人承担；也可以先以该分支机构管理的财产承担，不足以承担的，由法人承担
- **本书认为**：已登记、能以自己名义从事民事活动的分支机构，属于非法人组织

#### 第二节 法人的分类

**【课本原文】三大分类维度：**

**分类对比表：**

```
                        法人分类
                          │
          ┌───────────────┼───────────────┐
          │               │               │
    公法人 vs 私法人   营利 vs 非营利   社团 vs 财团
    （传统大陆法）    （《民法典》创新）  （传统学理）
```

**一、公法人与私法人**

| | 公法人 | 私法人 |
|---|--------|--------|
| 设立依据 | 依公法设立 | 依私法设立 |
| 目的 | 行使公共行政职能 | 一般不履行公共行政职能 |
| 举例 | 机关法人（县、市等） | 公司、企业等 |

> 【课本立场】采纳综合判断标准——通过考察设立的法律依据、事业目的等综合判断。

**二、营利法人与非营利法人（《民法典》三分法）**

| | 营利法人 | 非营利法人 |
|---|---------|-----------|
| 概念 | 以取得利润并分配给股东等出资人为目的（第76条第1款） | 为公益或其他非营利目的成立，不向出资人/设立人分配利润（第87条第1款） |
| 设立依据 | 特别法（公司法等） | 特别法（慈善法、基金法等） |
| 设立原则 | 准则主义为主 | 许可主义为主 |
| **核心区别** | 利润分配给出资人 | 利润归属于法人 |
| 终止后剩余财产 | 分配给成员 | 公益法人不得分配，非公益可分配 |
| 举例 | 有限责任公司、股份有限公司 | 事业单位、社会团体、基金会 |

**三、社团法人与财团法人**

| | 社团法人 | 财团法人 |
|---|---------|---------|
| 成立基础 | 社员（人的集合） | 财产（捐助行为） |
| 目的 | 可为营利或公益 | 只能为公益 |
| 设立程序 | 符合法定条件即可 | 一般需主管机关许可 |
| 设立人地位 | 设立后取得社员资格 | 设立后与法人脱离关系 |
| 举例 | 公司、行业协会 | 基金会、寺庙 |

> 《民法典》未采纳社团/财团分类，但捐助法人（基金会等）实质上属于财团法人。

#### 第三节 法人的民事权利能力和民事行为能力

**一、法人的民事权利能力**

**【课本原文】** 法人的民事权利能力，是指法人作为民事权利主体，享有民事权利并承担民事义务的资格。

- 《民法典》第59条：权利能力从法人成立时产生，到法人终止时消灭

**权利能力的特殊性（三点）：**

1. 不享有与公民的人身不可分离的权利（如生命权、健康权），以性别/年龄/身份/亲属关系等为前提的权利义务也不能享有和承担。**但可以享有名称权、荣誉权、名誉权等人格权。**
2. 依法受法律、行政法规的限制。
3. 受其章程和目的的限制。

**二、法人的民事行为能力**

**【课本原文→课本立场】** 我国采纳机关说（基于实在说）：法人具有自身的独立意志，通过法人的机关来实现。

**与自然人行为能力的区别：**

| | 法人 | 自然人 |
|---|------|--------|
| 时间 | 与权利能力同时产生、同时消灭 | 受年龄、健康影响，有权利能力不一定有行为能力 |
| 范围 | 与权利能力一致 | 可能小于权利能力 |
| 意思基础 | 区别于单个自然人的团体意思 | 个人意思 |

**目的范围对法人能力的限制效力——四种学说：**

| 学说 | 观点 |
|------|------|
| 权利能力限制说 | 目的范围限制的是权利能力 |
| 行为能力限制说 | 目的范围限制的是行为能力 |
| 代表权限制说 | 目的范围仅划定代表权范围 |
| 内部责任说 | 目的范围仅决定内部责任 |

> **【课本立场】** 区分类型：机关法人→权利能力限制；营利法人→行为能力限制。营利法人超越经营范围的行为不能简单认定无效（保护交易安全）。

**三、法人的侵权行为能力**

- 《民法典》第62条第1款：法定代表人因执行职务造成他人损害的，由法人承担民事责任
- **我国肯定法人具有侵权行为能力**

#### 第四节 法人的设立与登记

**一、设立方式五种（历史比较）**

| 方式 | 内容 |
|------|------|
| 自由主义（放任主义） | 具备法人实质即承认（18-19世纪，现基本放弃） |
| 特许主义 | 专门颁布法律成立特定法人 |
| 核准主义（许可主义） | 须经行政机关许可 |
| **准则主义（登记主义）** | 符合法定条件即可登记成立 |
| 强制主义 | 国家强制设立 |

> 我国营利法人原则上采**准则主义**，捐助法人/宗教场所法人等采**核准主义**。

**二、设立条件（《民法典》第58条第2款）**

法人应当有自己的：
1. **名称**——区别于其他法人的标志，法人对登记注册的名称享有专用权
2. **组织机构**——决策机构（股东会）+ 执行机构（董事会）+ 监督机构（监事会）
3. **住所**——以其主要办事机构所在地为住所（第63条）
4. **财产或者经费**——独立财产是其承担民事责任的物质基础

**三、登记效力**

- 登记要件主义：法人经登记成立
- 登记事项变更 → 应及时办理变更登记
- 《民法典》第65条：法人的实际情况与登记的事项不一致的，**不得对抗善意相对人**

#### 第五节 法人的法定代表人

**一、法定代表人的概念**

> **【课本原文·法条】** 《民法典》第61条第1款：依照法律或者法人章程的规定，代表法人从事民事活动的负责人，为法人的法定代表人。

**法定代表人三个特征：**
1. 由法律或法人章程规定的自然人
2. 有权代表法人从事民事活动（不需要特别授权）
3. 以法人名义从事的活动，由法人承担法律后果（第61条第2款）

**二、法定代表人的责任**

- 因**执行职务**造成他人损害的，由法人承担民事责任（第62条第1款）
- 执行职务的认定标准 → **外观说（客观说）**：社会一般人会将其理解为执行职务的行为即为职务行为
- 法人承担责任后，可向有过错的法定代表人追偿（第62条第2款）

**三、对法定代表人代表权的限制**

> **【法条】** 《民法典》第61条第3款：法人章程或者法人权力机构对法定代表人代表权的限制，不得对抗善意相对人。

适用条件：
1. 法人章程或权力机构对代表权作出了限制
2. 相对人是善意的（不知道或不应当知道该限制）

#### 第六节 法人的终止

> **【课本原文·法条】** 《民法典》第68条第1款：有下列原因之一并依法完成清算、注销登记的，法人终止：（一）法人解散；（二）法人被宣告破产；（三）法律规定的其他原因。

**终止流程图：**

```
解散/破产 ──→ 清算 ──→ 注销登记 ──→ 法人终止
```

**一、解散事由（第69条）：**
1. 章程规定的存续期间届满或解散事由出现
2. 权力机构决议解散
3. 因合并或分立需要解散
4. 依法被吊销营业执照/登记证书、被责令关闭或撤销
5. 法律规定的其他情形

**二、清算**

- 清算期间法人存续，但不得从事与清算无关的活动（第72条第1款）
- **清算义务人**：法人的董事、理事等执行机构或决策机构的成员（第70条第2款）
- 清算义务人未及时履行清算义务造成损害的 → 承担民事责任
- **清算法人的性质**：【课本立场】采**同一人格说**——清算前后的法人具有同一人格

**清算组的职权：**
- 对内：清理财产、整理账目、编造资产负债表
- 对外：收取债权、清偿债务
- 变卖财产
- 财产不足时申请宣告破产

**三、注销登记**

- 清算结束 + 完成注销登记 → 法人终止（第72条第3款）

#### 第七章 营利法人

**一、营利法人的概念（《民法典》第76条第1款）**

> **【课本原文】** 营利法人是指以取得利润并分配给股东等出资人为目的成立的法人。

**营利法人的四个特点：**
1. 社团法人的一种类型
2. 设立目的是分配利润给出资人（与是否营利本身不同！）
3. 主要从事经营活动
4. 终止后将剩余财产分配给成员

**二、营利法人的类型**

```
               营利法人
                  │
    ┌─────────────┼─────────────┐
    │             │             │
 有限责任公司   股份有限公司   其他企业法人
    │
    ├── 一般有限责任公司
    ├── 国有独资公司
    └── 一人有限责任公司
```

**三、营利法人的组织机构（三权分立）**

| 机构 | 功能 | 组成 |
|------|------|------|
| **权力机构** | 修改章程、选举更换执行/监督机构成员 | 股东会/股东大会 |
| **执行机构** | 执行决议、日常管理 | 董事会/执行董事 |
| **监督机构** | 监督执行机构活动 | 监事会/监事 |

**四、营利法人的特殊规则**

> **【课本原文】** 《民法典》第83条第1款：营利法人的出资人不得滥用出资人权利损害法人或者其他出资人的利益。

1. **法人人格否认（揭开公司面纱）**：不得滥用法人独立地位和有限责任损害债权人利益
2. **关联交易规则**：《民法典》第84条：不得利用关联关系损害法人利益，造成损失应赔偿
3. **决议撤销**：内容/程序违反法律或章程 → 出资人可申请撤销

### 🔑 核心知识点精要

#### 第三节 危害结果

**一、危害结果的概念**

> **【课本原文】** 广义的危害结果：犯罪行为对刑法所保护的社会关系所造成的损害——包括构成要件结果和非构成要件结果、物质性危害结果和非物质性危害结果、直接结果和间接结果等。

> **【课本原文】** 狭义的危害结果（构成要件结果）：由犯罪的实行行为造成的、根据刑法分则的规定对成立犯罪或者犯罪既遂具有决定意义的危害结果——只存在于过失犯罪、间接故意犯罪和结果犯的既遂犯中。

**【课本立场】** 危害结果不是一切犯罪构成的必要要件。

**二、危害结果的四种分类**

```
              危害结果分类
                  │
    ┌──────┬──────┼──────┬──────┐
    │      │      │      │      │
 (一)    (二)    (三)    (四)
 构成   物质   直接   基本
 要件   性与   结果   结果
 与非   非物   与间   与加
 构成   质性   接结   重结
 要件          果     果
```

**（一）构成要件结果与非构成要件结果**

| | 构成要件结果 | 非构成要件结果 |
|---|------------|--------------|
| 别称 | 定罪结果 | 量刑结果 |
| 意义 | 对定罪具有决定意义 | 对量刑具有评价意义 |
| 包含 | — | 行为犯结果、预备/未遂/中止犯结果、间接结果、精神损害 |

**（二）物质性危害结果与非物质性危害结果**

| | 物质性（有形） | 非物质性（无形） |
|---|-------------|---------------|
| 特征 | 具体、可见、可计量 | 抽象、不可直接测量 |
| 发生时机 | 一般在实行行为实施完毕后才发生 | — |
| 举例 | 死亡、伤害、财产损害 | 名誉受损、人格侮辱、威信受损 |

**（三）直接结果与间接结果**

| | 直接结果 | 间接结果 |
|---|---------|---------|
| 定义 | 危害行为本身直接引起 | 由直接结果进一步引起 |
| 与构成要件关系 | 构成要件结果只能是直接结果 | 只能是量刑情节 |

**（四）基本结果与加重结果**

| | 基本结果 | 加重结果 |
|---|---------|---------|
| 定义 | 作为犯罪构成要件的结果 | 引起另一种严重危害结果 |
| 法律效果 | 基本构成的要件 | 加重构成的要件（结果加重犯） |
| 注意 | — | 以刑法明文规定为限，不发生未遂问题 |

#### 第四节 刑法上的因果关系 ⭐

**一、概念**

> **【课本原文】** 刑法上的因果关系，是指人的危害行为合乎规律地引起某种危害结果的内在联系。

**二、因果关系的特点（六项）**

| 特点 | 核心内容 | 关键词 |
|------|---------|--------|
| **客观性** | 不以人的意志为转移的客观存在；不能凭司法人员臆测确定 | "客观存在" |
| **相对性** | 必须从普遍联系的链条中抽取危害行为与危害结果这对现象 | "抽取一对现象" |
| **时间顺序性** | 原因在先，结果在后，不可颠倒；但在先行为不一定都是原因 | "先因后果" |
| **条件性** | 因果关系是具体的、有条件的，不能脱离案件具体条件 | "具体条件" |
| **复杂性** | 一因多果（一个行为引起多种结果）、一果多因（多个原因引起一个结果） | "一因多果/一果多因" |
| **必然性和偶然性** | 必然因果关系与偶然因果关系是两种表现形式，因果关系本身是必然性和偶然性的统一 | "必然+偶然统一" |

**三、因果关系的性质**

> **【课本原文】** 刑法上因果关系的性质，是指危害行为在特定具体条件下合乎规律地引起了危害结果的发生。具体包括两方面：

1. **作为原因的危害行为必须具有引起危害结果发生的现实可能性**——因果之间具有质的同一性，原因包含着引起结果发生的根据和内容
2. **作为原因的危害行为必须合乎规律地引起危害结果的发生**——结果必须具有回避可能性；结果是具体的（特定样态、规模、时间、地点）

**四、因果关系的认定（五种特殊类型）**

| 类型 | 定义 | 认定规则 |
|------|------|---------|
| **假定的因果关系** | 即使没有该行为，其他情况也会产生同样结果 | **仍应肯定因果关系的存在**（课本立场） |
| **重叠的因果关系** | 两个以上独立行为单独不能导致结果，合并导致结果 | 均肯定因果关系 |
| **流行病学的因果关系** | 公害犯罪（污染环境等），排污行为与污染结果之间 | 必要条件（无A则无B）+ 病因学旁证法（四条件） |
| **不作为犯的因果关系** | 不履行作为义务与危害结果的关系 | 原因力在于"应该阻止而没有阻止事物向危险方向发展" |
| **中断的因果关系** | 因果关系进程中介入其他原因，切断原来因果进程 | **综合三因素**：①实行行为导致结果的可能性高低 ②介入因素异常性大小 ③介入因素对结果的贡献大小 |

**五、因果关系与刑事责任的关系**

> **【课本原文】** 存在因果关系 ≠ 必然承担刑事责任，还需具备主观罪过。

| 概念 | 口诀/比喻 | 记法解释 |
|------|----------|---------|
| 危害结果四个分类 | **"构件基物"** （构成要件→物质性→直接→基本，排列记忆） | 每个维度二分法：构/非构、物/非物、直接/间接、基本/加重 |
| 因果关系六特点 | **"客相时序条复必"** | 客观性→相对性→时间顺序性→条件性→复杂性→必然偶然性 |
| 中断的因果关系三因素 | **"三因素：可能+异常+贡献"** | 可能性高低+异常性大小+贡献大小 |
| 不作为的原因力 | **"应阻未阻"** | 应该阻止危险发展，却没有阻止 |

### 🔑 核心知识点精要

#### 一、三段论的特征和组成

**三段论**：由两个包含共同词项的性质命题推出一个新的性质命题的演绎推理。

```
大前提：所有 M 是 P        （包含大项P的命题）
小前提：所有 S 是 M        （包含小项S的命题）
─────────────────────
结  论：所有 S 是 P         （推出S与P的关系）
```

**三个词项：**

| 词项 | 符号 | 位置 | 说明 |
|------|------|------|------|
| **大项** | P | 大前提的谓项 + 结论的谓项 | 结论的谓项 |
| **中项** | M | 大前提和小前提中都出现 | **连接大小前提的关键** ⭐ |
| **小项** | S | 小前提的主项 + 结论的主项 | 结论的主项 |

#### 二、三段论的规则 ⭐⭐

> 六条关键规则：

| # | 规则 | 举例（违反则无效） |
|---|------|-------------------|
| 1 | **中项必须至少在一个前提中周延** | "有些M是P，有些S是M" → 中项M两次不周延 ❌ |
| 2 | 前提中不周延的词项在结论中不得周延 | 前提中"S是P的种"→结论不能说"所有S是P" |
| 3 | 两个否定前提不能得出结论 | M不是P，S不是M → 推不出S和P的关系 |
| 4 | 前提有一否定，结论必否定 | M是P，S不是M → 结论应为否定 |
| 5 | 两个特称前提不能得出结论 | "有些M是P，有些S是M" → 推不出 |
| 6 | 前提有一特称，结论必特称 | "所有M是P，有些S是M" → 结论："有些S是P" |

#### 三、三段论的格和式

**格：由中项在前提中的位置决定（四种格）**

| 格 | 中项位置 | 结构 | 常用场景 |
|----|---------|------|---------|
| 第一格 | 大前提主项、小前提谓项 | M-P, S-M → S-P | **法律适用最常用** ⭐ |
| 第二格 | 两前提谓项 | P-M, S-M → S-P | 区别格 |
| 第三格 | 两前提主项 | M-P, M-S → S-P | 例证格 |
| 第四格 | 大前提谓项、小前提主项 | P-M, M-S → S-P | 很少用 |

**式：由A、E、I、O四种命题的组合决定**

#### 四、三段论的省略式

在自然语言中，三段论常省略一个部分：

| 省略类型 | 何时使用 | 示例 |
|---------|---------|------|
| 省略大前提 | 法律依据默认已知 | "你是法人，所以你有民事责任能力"（省略"凡法人都有民事责任能力"） |
| 省略小前提 | 案件事实不言自明 | "凡法人都应依法纳税，所以该公司应依法纳税"（省略"该公司是法人"） |
| 省略结论 | 不言而喻 | "凡公民都有肖像权，张三是公民"（省略"张三有肖像权"） |

**揭示省略部分的方法**：根据上下文判断哪个命题缺失，用逻辑规则还原。

## 💡 今日跨科串联

> 🧵 **"因果关系的逻辑结构"** —— 今天三科有一个共同的暗线：
>
> - **逻辑学**教的是"三段论"的形式结构：大前提→小前提→结论
> - **刑法学**教的是"因果关系"的实质判断：行为是否合乎规律地引起结果
> - **民法学**教的是"法人责任"的归属规则：谁的行为→谁承担后果
>
> 三者的共同结构都是 **"原因→结果"** / **"前提→结论"** 的推理链条。刑法的因果关系判断本质上就是在运用逻辑的因果推理，而民法中法定代表人"执行职务"的认定也是因果归属问题。
