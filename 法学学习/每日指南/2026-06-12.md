### 🔑 核心知识点精要

#### 第一节 民事法律行为生效的概念和特征

**【课本原文】生效概念**
> 所谓民事法律行为的生效，是指已经成立的法律行为因符合法定的生效要件，从而能产生法律上的约束力。

**【课本原文】法条依据**
> 《民法总则》第136条第1款："民事法律行为自成立时生效，但是法律另有规定或者当事人另有约定的除外。"

##### 成立 vs 生效 —— 六大区别

| 维度 | 成立 | 生效 |
|------|------|------|
| ① 性质 | 涉及当事人个人意思的问题（自主自愿） | 法律对意思表示作出**评价或干预** |
| ② 阶段 | 意思表示达成合意 | 产生实质法律拘束力，可能晚于成立（如须审批） |
| ③ 要件 | 当事人+意思表示+合意 | 第143条三要件（行为能力+意思真实+合法） |
| ④ 国家干预 | 主要体现当事人意思 | 体现国家**肯定或否定评价**，反映国家干预 |
| ⑤ 逻辑体系 | 区分不成立/被撤销/无效，**事实判断vs价值判断** | — |
| ⑥ 法律后果 | 不成立→缔约过失责任（仅民事责任） | 无效→民事责任+可能行政/刑事责任 |

> 📌 **课本立场**：《民法总则》第136条采"成立即生效"为原则，法律另有规定或当事人另有约定除外。

#### 第二节 民事法律行为的生效要件 ⭐⭐⭐

**【课本原文】第143条三要件**
> 《民法总则》第143条："具备下列条件的民事法律行为有效：（一）行为人具有相应的民事行为能力；（二）意思表示真实；（三）不违反法律、行政法规的强制性规定，不违背公序良俗。"

##### 要件一：行为人具有相应的民事行为能力

| 民事行为能力类型 | 可独立实施的范围 |
|-----------------|-----------------|
| 完全民事行为能力人 | 所有民事法律行为 |
| 限制民事行为能力人 | ①纯获利益的行为 ②与年龄/智力/精神健康状况相适应的行为（其他须法定代理人同意或追认，第145条第1款） |
| 无民事行为能力人 | 不能独立实施（第144条：无效）；法定代理人代理 |

> 📌 法人超越经营范围：不当然无效，须衡量相对人善意与否。但违反**国家限制经营/特许经营/禁止经营**的除外。

##### 要件二：意思表示真实

> 意思表示真实，是指表意人的表示行为应当真实地反映其内心的效果意思。即**表示行为应当与效果意思相一致**。

意思表示不真实的两种情况：

| 类型 | 含义 | 示例 |
|------|------|------|
| 意思表示不自由 | 因欺诈/胁迫等外在原因导致意志不自由 | 被欺骗签合同 |
| 意思表示不一致 | 外部表达不符合内心真实意思 | 通谋虚伪表示（名为赠与实为逃债） |

##### 要件三：不违反强制规定 + 不违背公序良俗

```
不违反法律、行政法规的强制性规定
├── "法律"=全国人大及其常委会制定
├── "行政法规"=国务院制定
├── 只限强制性规定（非任意性规定）
└── 效力性强制规定 vs 管理性强制规定
    ├── 违反效力性强制规定 → 无效
    └── 违反管理性强制规定 → 不一定无效（第153条但书）

不违背公序良俗
├── 公共秩序：弥补强制性规定的不足（兜底作用）
└── 善良风俗：社会公共道德
```

#### 第三节 效力待定的民事法律行为 ⭐

**【课本原文】概念**
> 效力待定的民事法律行为，是指法律行为成立之后，是否能发生效力尚不能确定，有待于其他行为或事实使之确定的法律行为。

##### 三特征

| 特征 | 说明 |
|------|------|
| ① 已成立但效力不齐备 | 因缺乏处分权或行为能力 |
| ② 效力不确定的中间状态 | 既非完全无效，也非完全有效 |
| ③ 有待其他行为/事件确定 | 体现私法自治，不实行国家干预 |

##### 三种类型

| 类型 | 法条依据 |
|------|---------|
| ① 限制民事行为能力人超越能力范围 | 第145条第1款 |
| ② 无权代理 | 第171条第1款（不含表见代理） |
| ③ 无权处分 | 《合同法》第51条（善意取得可补正） |

##### 效力确定：追认 vs 撤销

```
相对人的权利：
├── 催告权：催告法定代理人/被代理人30日内追认
│   └── 期满不表示 → 视为拒绝追认
└── 撤销权（善意相对人）：追认前以通知方式撤销

追认权人的权利：
└── 追认：单方意思表示，无须相对人同意
```

#### 第四节 无效民事法律行为 ⭐⭐⭐

**【课本原文】概念与四特征**
> 无效民事法律行为，是指虽然已经成立，但因其在内容上违反了法律、行政法规的强制性规定和公序良俗而应当被宣告无效的民事行为。

| 特征 | 关键词 |
|------|--------|
| ① 违法性 | 违反强制规定+公序良俗 |
| ② 国家干预 | 法院/仲裁机构**主动审查**，不待当事人请求 |
| ③ 不得履行性 | 不得请求实际履行，不承担不履行的法律责任 |
| ④ 自始无效 | 溯及力，自成立时起无效 |

##### 五种无效类型

| 类型 | 法条 | 要点 |
|------|------|------|
| ① 无民事行为能力人实施 | 第144条 | 全部无效（但纯获利益行为依《民通意见》第6条继续有效） |
| ② 虚假的民事法律行为（通谋虚伪表示） | 第146条第1款 | 双方共同虚假表示→无效；隐藏行为依相关法律规定处理 |
| ③ 违反强制性规定 | 第153条第1款 | 限于法律/行政法规的效力性强制规定 |
| ④ 违背公序良俗 | 第153条第2款 | 危害公共秩序+违背善良风俗（如赌债合同、生死条款） |
| ⑤ 恶意串通损害他人 | 第154条 | 当事人互相串通+损害特定第三人→由受害第三人主张无效 |

##### ⭐ 虚假行为 vs 恶意串通 对比

| | 通谋虚伪表示 | 恶意串通 |
|------|------------|---------|
| 目的 | 不以损害第三人为必要 | 以损害第三人利益为目的 |
| 重点 | 意思表示的**虚假** | 双方的**串通** |
| 主张主体 | 任何人可主张 | 通常由受害第三人主张 |

##### 部分无效规则
> 《民法总则》第156条："民事法律行为部分无效，不影响其他部分效力的，其他部分仍然有效。"

前提：无效部分与有效部分具有**可分性**，且不与当事人愿望相违背。

#### 第五节 可撤销的民事法律行为 ⭐⭐⭐

**【课本原文】概念**
> 可撤销的民事法律行为，是指当事人在从事民事法律行为时，因意思表示不真实，法律允许撤销权人通过行使撤销权而使该已经生效的法律行为归于无效。

##### 三特征

| 特征 | 说明 |
|------|------|
| ① 主要是意思表示不真实 | 撤销的是意思瑕疵 |
| ② 须撤销权人主动行使 | 法院**不告不理** |
| ③ 撤销前完全有效 | 不同于效力待定和无效 |

##### 四种可撤销类型

```
可撤销的民事法律行为
├── ① 重大误解（第147条）
│   ├── 条件：对内容有重大误解 + 因误解而作出意思表示 + 误解由自身过错造成
│   └── 《民通意见》第71条：对行为性质/对方当事人/标的物品种/质量/规格/数量的错误认识
├── ② 欺诈（第148-149条）
│   ├── 一方欺诈：故意告知虚假情况或故意隐瞒真实情况→对方陷入错误→作出意思表示
│   └── 第三人欺诈：对方知道或应当知道该欺诈行为时→可撤销
├── ③ 胁迫（第150条）
│   ├── 一方或第三人实施胁迫
│   ├── 以将来损害或直接损害相威胁→产生恐惧→作出意思表示
│   └── 胁迫行为必须非法（合法手段如诉讼威胁不构成胁迫）
└── ④ 显失公平（第151条）
    ├── 客观要件：给付严重失衡（必须成立时即显失公平）
    └── 主观要件：利用对方危困状态/缺乏判断能力/优势地位
```

##### 撤销权行使规则

| 事项 | 内容 |
|------|------|
| 主体 | 因意思表示不真实而受损害的一方（欺诈/胁迫人不可撤销） |
| 方式 | 须以**诉讼或仲裁**方式 |
| 性质 | 形成权，专属权，不可单独转让 |
| 变更权 | **不包含变更权**（《民法总则》未规定变更权，已修改《合同法》第54条） |

##### 撤销权除斥期间

| 情形 | 起算点 | 期限 |
|------|--------|------|
| 一般（欺诈、显失公平等） | 自知道或应当知道撤销事由之日 | **1年** |
| 重大误解 | 自知道或应当知道撤销事由之日 | **3个月** |
| 胁迫 | 自胁迫行为终止之日 | **1年** |
| 最长客观期间 | 自民事法律行为发生之日 | **5年** |

> 📌 撤销权可放弃（明示或默示），放弃后消灭。

##### 被撤销后的法律后果（第157条）

```
民事法律行为无效/被撤销/确定不发生效力后
├── 返还财产（原物+孳息）
│   └── 不能返还或没必要返还 → 折价补偿
├── 赔偿损失（信赖利益损失）
│   ├── 单方过错 → 有过错方赔偿
│   └── 双方过错 → 各自承担相应责任
└── 自始没有法律约束力（第155条）
```

#### 第六节 附条件和附期限的民事法律行为

##### 条件 vs 期限

| | 条件 | 期限 |
|------|------|------|
| 性质 | **不确定**是否发生 | **确定**必然到来 |
| 作用 | 限制效力发生/消灭 | 限制效力发生/消灭 |

##### 条件六特征

```
条件必须具备的条件（缺一不可）
├── ① 将来发生的事实（非过去事实）
├── ② 不确定的事实（非必然发生）
├── ③ 条件成就必须可能（非不可能）
├── ④ 当事人约定（非法定条件）
├── ⑤ 合法（不违反法律和公序良俗）
└── ⑥ 不得与民事法律行为主要内容矛盾
```

##### 条件分类

| 分类标准 | 类型 |
|---------|------|
| 对效力作用 | **停止条件**（生效条件）→ 条件成就时生效 |
| | **解除条件**（消灭条件）→ 条件成就时失效 |
| 事实发生 | **积极条件**（某种事实发生） |
| | **消极条件**（某种事实不发生） |
| 成就原因 | 偶成条件 / 随意条件 / 混合条件 |

##### 恶意促成/阻止的拟制（第159条）
- 恶意促成条件成就 → **视为条件不成就**
- 恶意阻止条件成就 → **视为条件已成就**

##### 期待权保护
在条件成就与否的"未决期间"，当事人享有**期待权**，受法律保护。第三人侵害期待权应负损害赔偿责任。

##### 期限分类

| 类型 | 效力 | 例 |
|------|------|-----|
| 始期（生效期限） | 期限届至→生效 | "自2025年10月1日起生效" |
| 终期（终止期限） | 期限届满→失效 | "至2026年12月31日终止" |

### 🧠 记忆诀窍

**1. 效力状态光谱口诀："有效无效看要件，待定等人来拍板，可撤有人喊不算"**

```
全部有效 ── 可撤销 ── 效力待定 ── 无效
   │          │         │          │
  要件齐      意思瑕疵   能力/权源缺  违法违俗
  (三要件)    (可喊撤)   (等人追认)  (自始无效)
```

**2. 可撤销四类型口诀："大欺胁公"**
- **大** = 重大误解（3个月）
- **欺** = 欺诈（一方+第三人，1年）
- **胁** = 胁迫（1年，自胁迫终止起算）
- **公** = 显失公平（利用危困/缺判断力，1年）

**3. 无效五种类型速记："无能假强公恶"**
- 无 → 无民事行为能力人
- 假 → 虚假表示（通谋虚伪）
- 强 → 违反强制性规定
- 公 → 违背公序良俗
- 恶 → 恶意串通

**4. 成立vs生效一句话总结**："成立是私人合意，生效是国家盖章认证"

**5. 条件vs期限一句话**："条件可能不发生，期限迟早会到来"

### 🔑 核心知识点精要

#### 一、第10章深化：既遂类型与未遂分类

##### 犯罪既遂的四种类型

| 类型 | 既遂标准 | 举例 |
|------|---------|------|
| **行为犯** | 只要实施完毕某种行为即既遂 | 诬告陷害罪 |
| **结果犯** | 须发生法定危害结果才既遂 | 故意杀人罪 |
| **危险犯** | 造成法定的危险状态即既遂 | 放火罪（足以危害公共安全） |
| **举动犯** | 一经着手实行即既遂 | 煽动分裂国家罪 |

##### 犯罪未遂的两种分类

```
犯罪未遂的分类
├── 以实行行为是否终了
│   ├── 实行终了的未遂：已实施完毕但结果未发生
│   └── 未实行终了的未遂：尚未实施完毕即被阻止
└── 以行为能否实际达到既遂
    ├── 能犯未遂：行为本身有可能完成犯罪（如开枪未击中）
    └── 不能犯未遂：行为本身不可能完成
        ├── 工具不能犯（如用空枪杀人）
        └── 对象不能犯（如误将尸体当活人射击）
```

> ⚠️ 中止"自动性"的关键判断：**弗兰克公式**——"能达目的而不欲"=中止，"欲达目的而不能"=未遂。

#### 二、共同犯罪概述（第11章第一节）

**【课本原文】概念**
> 《刑法》第25条："共同犯罪是指二人以上共同故意犯罪。二人以上共同过失犯罪，不以共同犯罪论处；应当负刑事责任的，按照他们所犯的罪分别处罚。"

##### 共同犯罪的成立三要件

| 要件 | 内容 |
|------|------|
| **主体要件** | 二人以上（自然人或单位），均达到刑事责任年龄、具有刑事责任能力 |
| **客观要件** | 共同的犯罪行为（分工协作、互相配合，指向同一目标） |
| **主观要件** | 共同的犯罪故意（有意思联络，明知共同行为会发生危害结果，希望或放任） |

##### 不构成共同犯罪的情形（六种）

| 情形 | 处理方式 |
|------|---------|
| ① 共同过失犯罪 | 按各自所犯罪行**分别处罚**（第25条第2款） |
| ② 一方故意 + 一方过失 | 不构成共同犯罪 |
| ③ 无意思联络的同时犯 | 各自单独犯罪 |
| ④ 超出共同故意的**过限行为** | 过限部分由实施者单独负责 |
| ⑤ 事前无通谋的窝藏、包庇 | 分别构成窝藏/包庇罪（第310条） |
| ⑥ 片面共犯 | **本书认为**不构成共同犯罪（因无双向意思联络） |

> 📌 **课本立场（片面共犯）**："如果所谓片面共犯就是片面共同犯罪，则片面共犯的提法本身就是自相矛盾的。"

---

#### 三、共同犯罪的形式（第11章第二节）

##### 四种分类体系

```
共同犯罪的形式
├── ㈠ 以是否可由一人实施
│   ├── 任意共同犯罪：可由一人也可由多人实施的犯罪
│   └── 必要共同犯罪：必须二人以上（聚众性/集团性）
│       ├── 聚众性共同犯罪：首要分子组织策划指挥，三人以上
│       └── 集团性共同犯罪：犯罪集团实施
│
├── ㈡ 以共同故意形成时间
│   ├── 事前通谋：着手前已形成共同故意（社会危害性更大）
│   └── 事前无通谋（事中通谋）：着手时或实行过程中形成
│
├── ㈢ 以有无分工
│   ├── 简单共犯：所有人都是实行犯
│   └── 复杂共犯：有分工（组织犯/教唆犯/帮助犯/实行犯）
│
└── ㈣ 以有无组织形式
    ├── 一般共犯：松散，无固定组织形式
    └── 犯罪集团（有组织共犯）：三人以上，较为固定
        ├── 成员多数性（三人以上）
        ├── 共同实施犯罪的目的性
        ├── 较强的组织性
        └── 相当的稳固性（为长期犯罪而联合）
```

#### 四、共同犯罪人及其刑事责任（第11章第三节）⭐⭐⭐

##### 四种共犯人类型速览

| 类型 | 法条 | 定义 | 刑事责任 |
|------|------|------|---------|
| **主犯** | 第26条 | ①犯罪集团首要分子 ②共同犯罪中起主要作用者 | 首要分子→集团全部罪行；其他主犯→所参与/组织/指挥的全部犯罪 |
| **从犯** | 第27条 | ①次要实行犯 ②帮助犯 | **应当**从轻、减轻或免除处罚 |
| **胁从犯** | 第28条 | 被胁迫参加犯罪（排除被诱骗者） | 按犯罪情节**减轻或免除**处罚 |
| **教唆犯** | 第29条 | 故意唆使他人犯罪 | 按在共犯中的作用处罚；教唆不满18周岁→从重处罚 |

##### 主犯定罪范围详解

```
主犯的刑事责任范围
├── 犯罪集团首要分子（第26条第3款）
│   └── 按照集团所犯的全部罪行处罚
│       ├── "集团所犯的全部罪行" ≠ "集团成员所犯的全部罪行"
│       ├── 过限行为不属于集团罪行（首要分子不负责）
│       └── 课本立场：经过修正的"预谋罪行负责说"（集团故意负责说）
│
└── 其他主犯（第26条第4款）
    └── 按照其所参与的或者组织、指挥的全部犯罪处罚
```

##### ⚠️ 教唆犯特殊规则

| 情形 | 处罚 |
|------|------|
| 教唆他人犯罪（被教唆人已犯） | 按在共同犯罪中的**作用**处罚（通常为主犯） |
| 教唆**不满18周岁**的人犯罪 | **从重处罚** |
| 被教唆人未犯被教唆之罪（教唆未遂） | **可以从轻或减轻处罚** |
| 教唆不满12岁或已满12不满16（非特定犯罪） | 以**间接正犯**论处 |

##### ⚠️ 胁从犯 vs 紧急避险

> 当胁迫是正在发生的直接威胁，行为人为了保护较大利益而被迫损害较小利益 → 认定为**紧急避险**，不按胁从犯处理。

##### 胁从犯的转化
起初被胁迫，后来变积极 → 按转化后情况认定为从犯或主犯。

**1. 共同犯罪成立三要件口诀："两人有意思"**
- **两** = 二人以上（主体）

- 窝藏 → 事后窝藏包庇

```
主犯 > 从犯 > 胁从犯
 │       │       │

           或免除

教唆犯：看作用定主从 + 教唆未成年人从重 + 教唆未遂可减
```

**4. 犯罪集团四特征口诀："多目强固"**
- **多** = 成员多数性（三人以上）

### 🔑 核心知识点精要

#### 第一节 类比推理的特征及性质

**【课本原文】类比推理的定义**
> 根据两个（类）对象在某些属性上相同或相似，推出它们在另一些属性上也相同或相似的推理。

##### 类比推理的公式

```
已知：A具有属性a、b、c、d
      B具有属性a、b、c
结论：所以，B可能具有属性d
```

##### 类比推理的性质

| 性质 | 说明 |
|------|------|
| **推理方向** | 从特殊到特殊（或从一般到一般）——区别于演绎和归纳 |
| **结论性质** | **或然性**（可能真，不是必然真） |
| **可靠性因素** | ①共有属性的**相关性**（与推出属性的关联程度）②共有属性的**数量** |

##### 三种推理方式对比

| | 演绎推理 | 归纳推理 | 类比推理 |
|------|---------|---------|---------|
| 方向 | 一般→特殊 | 特殊→一般 | **特殊→特殊** |
| 结论 | 必然（前提真结论必真） | 或然 | 或然 |
| 可靠性 | 取决于形式有效性 | 取决于样本代表性 | 取决于共有属性相关性和数量 |

---

#### 第二节 类比推理在司法工作中的应用

##### 三大应用场景

| 应用 | 逻辑原理 | 实例 |
|------|---------|------|
| **类比并案** | 多起案件作案手法/特征相似→可能为同一人/同一伙人作案 | 系列抢劫案串并侦查 |
| **侦查实验类比** | 模拟案发条件实验→推断实际案件可能性 | 模拟犯罪过程验证嫌疑人供述 |
| **类比法律推理** | 类似案件类似处理 | 普通法系先例推理 / 中国指导性案例参照适用 |

---

#### 第三节 刑事侦查中的比对推理

##### 比对推理的特点
- 根据对象特征的相同点进行**同一认定**
- 应用领域：笔迹鉴定、指纹比对、DNA比对、弹道比对

##### 比对推理的逻辑要求

| 要求 | 含义 |
|------|------|
| **特征必须具有特异性** | 特征能实现个体识别（如DNA、指纹具有高度个体特异性） |
| **特征必须具有稳定性** | 特征在一定时期内保持不变（如指纹终身不变） |
| **结论的确定程度** | 取决于特征的质量（特异性高低）和数量（吻合点多少） |

---

### 🧠 记忆诀窍

**1. 类比推理一句话总结**："A有abc→B有ab，猜B也有c"

**2. 三种推理方向口诀："演归比，一般特，特一般，特特行"**

```
演绎：一般 → 特殊（必然）
归纳：特殊 → 一般（或然）
类比：特殊 → 特殊（或然）
```

**3. 比对推理两要件口诀："特稳"**
- **特** = 特征必须具有**特**异性（能区分个体）
- **稳** = 特征必须具有**稳**定性（能保持一段时间）

**4. 类比可靠性提高法**：相关属性越多 → 越可靠；共有属性与推出属性越相关 → 越可靠

> - 🏛️ 民法：法律行为的**效力**——有效/无效/效力待定/可撤销，核心是看"有没有瑕疵"（行为能力的瑕疵、意思表示的瑕疵、合法性的瑕疵）
