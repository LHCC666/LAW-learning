### 🔑 核心知识点精要

#### 第一节 意思表示概述

##### 一、意思表示的概念和特征

**【课本原文】** 所谓意思表示，是指向外部表明意欲发生一定私法上效果之意思的行为。意思表示是法律行为的核心。

**三个特征：**

| 特征 | 要点 |
|------|------|
| **第一** | 表意人具有旨在使法律关系发生变动的意图，不违反强制性规定和公序良俗 → 意思自治的工具 |
| **第二** | 意思由内到外的表示过程——单纯的内心意思没有法律意义，必须表示在外为他人知晓 |
| **第三** | 依是否符合生效要件，法律赋予不同效力——符合生效要件→预期效果；不符合→效果可能与内心意思不一致 |

##### 二、意思表示与民事法律行为的区别

| 区别维度 | 意思表示 | 民事法律行为 |
|----------|---------|-------------|
| **数量** | 只能是一方的意思表达 | 可由一个（单方）、两个（双方）或多个（多方）意思表示构成 |
| **成立要件** | 当事人 + 意思表示 + 标的 | 还可能有法定/约定特殊成立要件（书面/登记/公证） |
| **成立时间** | 通常以发出为准 | 单方：作出或到达；双方/多方：意思表示一致；要物行为：完成交付 |
| **生效** | 区分对话/非对话方式分别采了解主义/到达主义 | 依《民法总则》第143条：行为能力+意思表示真实+不违法不背俗 |

##### 三、意思表示的构成要素

**【课本原文】** 按照传统民法理论，意思表示的构成要素可概括为五种：

```
意思表示的构成要素
├── 主观要件（内在意思）
│   ├── 行为意思：自觉地从事某项行为的意思
│   ├── 表示意思：认识到其行为具有某种法律上的意义
│   ├── 目的意思（基础意思/交易意思）：表明法律行为具体内容
│   │   ├── 要素：必须具备的内容（如买卖合同的标的物、数量）
│   │   ├── 常素：通常应含有的内容（如承揽合同风险负担）
│   │   └── 偶素：依行为人特殊意志确定（如占有改定约定）
│   └── 效果意思（法效意思/效力意思）：欲使其表示引起法律上效力的内在意思
│       └── ⚠️ 与动机区分：动机≠效果意思（如"为方便读书租房"是动机，"取得房屋使用权"是效果意思）
└── 客观要件（外在表示）
    └── 表示行为：将效果意思表现于外部
        ├── 明示：将内在意思明确表达于外部
        └── 默示：未明确表示，但依法律规定可推知
```

> ⚠️ **效果意思 vs 目的意思**：目的意思指向意思表示的具体内容（如要约中的合同条款），效果意思指向意欲发生法律效果的意图（如"一经承诺即受合同拘束"）。

#### 第二节 意思表示的生效

##### 生效模式四种学说

| 学说 | 核心观点 | 我国立场 |
|------|---------|---------|
| 表意主义 | 意思决定具备外在形态即生效 | 适用于**无相对人**的意思表示 |
| 发送主义 | 不仅表现于外，还必须发出 | — |
| **到达主义** | 进入相对人可了解范围即生效（不要求实际了解） | 适用于**非对话方式**有相对人 |
| **了解主义** | 相对人必须实际了解 | 适用于**对话方式**有相对人 |

##### 意思表示生效的具体情形（《民法总则》第137-139条）

| 类型 | 生效规则 | 法条依据 |
|------|---------|---------|
| 以对话方式作出 | **了解主义**：相对人知道其内容时生效 | 第137条第1款 |
| 以非对话方式作出 | **到达主义**：到达相对人时生效 | 第137条第2款 |
| 数据电文（指定系统） | 进入该特定系统时生效 | 第137条第2款 |
| 数据电文（未指定系统） | 相对人知道或应当知道进入其系统时生效 | 第137条第2款 |
| 无相对人的意思表示 | **完成主义**：表示完成时生效 | 第138条 |
| 以公告方式作出 | 公告发布时生效 | 第139条 |

##### 意思表示的撤回（《民法总则》第141条）

> ⚠️ **撤回 ≠ 撤销**：撤回是使意思表示**不发生效力**（在生效前）；撤销是使**已经生效**的意思表示**失去效力**。我国《民法总则》仅规定撤回，未规定撤销。

**【课本原文】** 撤回意思表示的通知应当在意思表示到达相对人前或者与意思表示同时到达相对人。

#### 第三节 意思表示的形式（《民法总则》第140条）

| 形式 | 定义 | 示例 |
|------|------|------|
| **明示** | 将内在意思明确表达于外部（口头/书面） | 签约、发函 |
| **默示** | 未明确表示，但依法律规定可推知 | 点头、举手、起立 |
| **沉默** | 未作出任何表示或不行为 | **原则上不构成意思表示** |

**【课本原文】** 沉默只有在有法律规定、当事人约定或者符合当事人之间的交易习惯时，才可以视为意思表示。

> 📋 沉默视为意思表示的三种例外：
> 1. 法律特别规定（如《继承法》第25条：继承开始后未表示放弃→视为接受）
> 2. 当事人特别约定
> 3. 当事人之间存在交易习惯

#### 第四节 意思表示的解释（《民法总则》第142条）

##### 两种类型对比

| 类型 | 有相对人的意思表示 | 无相对人的意思表示 |
|------|-------------------|-------------------|
| 核心规则 | 按所用词句 + 结合条款/性质目的/习惯/诚信 | **不能完全拘泥于词句** + 探求真实意思 |
| 是否需要受领人 | ✅ 需要 | ❌ 不需要 |
| 是否考虑受领人理解水平 | ✅ 考虑一般受领人水平 | ❌ 不需要 |
| 信赖利益保护 | ✅ 考虑客观主义（外观法理） | ❌ 不考虑（探求真意为主） |

##### 五种解释规则

| 规则 | 法条依据 | 核心内容 |
|------|---------|---------|
| **1. 按通常理解解释** | 第142条第1款 | 以合理人标准理解用语含义 |
| **2. 整体解释（体系解释）** | 第142条"结合相关条款" | 各种资料综合考虑，不断章取义 |
| **3. 目的解释** | 第142条"行为的性质和目的" | 根据当事人所追求的目的来解释 |
| **4. 习惯解释** | 第142条"习惯" | 依交易习惯或生活习惯解释 |
| **5. 诚信原则解释** | 第142条"诚信原则" | 填补漏洞+平衡双方利益（一般最后适用） |

### 🧠 记忆诀窍

**口诀：意思表示五要素 → "行表目效示"**
- **行**为意思（自觉行为）
- **表**示意思（认知法律意义）
- **目**的意思（具体内容：要素/常素/偶素）
- **效**果意思（追求法效）
- **示**行为（外部表达：明示/默示）

**口诀：生效四主义 → "表发到了"**
- **表**意主义（无相对人）
- **发**送主义
- **到**达主义（非对话）
- **了**解主义（对话）

**一句话总结**：意思表示 = 内在意思（行表目效）+ 外在表示，是法律行为的核心。生效依有无相对人区分四种模式。沉默原则上≠意思表示。

### 🔑 核心知识点精要

#### 一、紧急避险（第9章第三节）

##### 概念与特征

**【课本原文】** 《刑法》第21条第1款："为了使国家、公共利益、本人或者他人的人身、财产和其他权利免受正在发生的危险，不得已采取的紧急避险行为，造成损害的，不负刑事责任。"

**三个特征：**
1. 通过对**合法**的人身、财产权益的损害行为实施
2. 在合法权益遭受重大危险时采取的避免危险的紧急措施，整体有益于社会
3. 是在**不得已**情况下采取的权宜之计

##### 六个成立条件（缺一不可）

```
紧急避险成立条件
├── 1. 起因条件：合法权益遭受损害危险
│   ├── 人的不法侵害行为
│   ├── 自然灾害（地震/火灾/洪水等）
│   ├── 动物的侵袭
│   └── 人的生理、疾患
│   ⚠️ 危险必须客观存在 → 假想避险按认识错误处理
├── 2. 时间条件：危险正在发生
│   ⚠️ 事前避险/事后避险 = 避险不适时，不构成紧急避险
├── 3. 对象条件：损害的是第三者合法权益
│   ⚠️ 若损害不法侵害人本人 → 是正当防卫，不是紧急避险
├── 4. 目的条件：主观上有正当的避险意图
│   → 出于避免国家/公共利益/本人或他人合法权益免受危险的目的
├── 5. 限度条件：不能超过必要限度
│   → 避险造成的损害 ＜ 所避免的损害（必须小于，不能等于或大于）
│   ⚠️ 人身权利 > 财产权利；财产权利按价值大小衡量
├── 6. 限制条件：必须在不得已的情况下实施
│   → 别无他法，只能通过损害较小权益来保全较大权益
│   ⚠️ 职务上/业务上负有特定责任的人，不能为避免本人危险进行紧急避险
│      （军人/消防员/民航机组等）
```

##### 避险过当

- **概念**：避险行为超过必要限度造成不应有的损害 → 应当负刑事责任
- **罪过形式**：通常为疏忽大意的过失；少数为间接故意或过于自信的过失；排除直接故意
- **定罪**：避险过当本身不是独立罪名，根据主观罪过+客观行为按分则条款定罪（如过失致人死亡罪）
- **量刑**："应当减轻或者免除处罚"（《刑法》第21条第2款）

##### 紧急避险 vs 正当防卫 五大区别

| 区别维度 | 正当防卫 | 紧急避险 |
|----------|---------|---------|
| **起因条件** | 仅限于**人的不法侵害** | 人的不法侵害 + 自然灾害 + 动物侵袭 + 生理疾患 |
| **对象条件** | 针对**不法侵害人本人** | 针对**第三者合法权益** |
| **限度条件** | 未明显超过必要限度 + 未造成重大损害 | 损害**必须小于**所避免的损害 |
| **限制条件** | 无特殊限制（正在进行的侵害即可） | **不得已**时才能实施 |
| **主体要求** | 无特殊限制 | 职务/业务上负有特定责任者不能为避免本人危险而实施 |

#### 二、其他正当行为（第9章第四节）

| 类型 | 核心含义 | 关键条件 |
|------|---------|---------|
| **自救行为** | 无法等待公权力救济时自力恢复权利 | 侵害已发生；来不及法律程序；有自救目的；行为相当 |
| **正当业务行为** | 基于职业要求实施的行为（医疗/竞技等） | 业务合法；有益社会的目的；未超必要限度 |
| **法令行为** | 依据法律法规或上级命令实施的合法行为 | 有法律依据/命令合法；有益目的；在限度内 |
| **被害人承诺** | 经权利人同意后损害其本身利益 | 权利人有权处理；同意自愿/合法/有益；在授权范围内 |

#### 三、故意犯罪的停止形态（第10章）⭐⭐⭐

##### 体系总览

```
故意犯罪停止形态（只存在于部分直接故意犯罪中）
├── 完成形态
│   └── 犯罪既遂（四种类型）
│       ├── 结果犯：以法定犯罪结果发生为既遂标志
│       ├── 行为犯：以法定犯罪行为完成为既遂标志
│       ├── 危险犯：以造成法定危险状态为既遂标志
│       │   ├── 具体危险犯：需司法上具体认定危险（紧迫危险）
│       │   └── 抽象危险犯：立法推定有危险（不紧迫）
│       └── 举动犯：一经着手实行即成立既遂
│
└── 未完成形态
    ├── 犯罪预备：为了犯罪准备工具/制造条件 + 意志以外原因停止在预备阶段
    │   → 处罚："可以比照既遂犯从轻、减轻或免除处罚"（《刑法》第22条）
    ├── 犯罪未遂：已着手实行 + 意志以外原因未得逞
    │   ├── 实行终了的未遂 / 未实行终了的未遂
    │   ├── 能犯未遂 / 不能犯未遂（对象不能/工具不能）
    │   → 处罚："可以比照既遂犯从轻或减轻处罚"（《刑法》第23条）
    └── 犯罪中止：犯罪过程中 + 自动放弃或自动有效防止结果发生
        ├── 自动停止型中止（预备阶段或实行阶段）
        └── 有效防止型中止（实行阶段，危害结果可能发生时）
        → 处罚："没有造成损害的应当免除；造成损害的应当减轻"（《刑法》第24条）
```

##### 四种停止形态核心特征对比

| 形态 | 所处阶段 | 停止原因 | 处罚原则 | 法条 |
|------|---------|---------|---------|------|
| **既遂** | 实行完成 | 构成要件齐备 | 按分则法定刑处罚 | — |
| **预备** | 预备阶段 | 意志以外原因 | **可以**从轻/减轻/**免除** | 第22条 |
| **未遂** | 已着手实行 | 意志以外原因 | **可以**从轻/**减轻** | 第23条 |
| **中止** | 全过程中 | **自动**放弃/自动有效防止 | **应当**免除（无损害）/**减轻**（有损害） | 第24条 |

> ⚠️ **关键区分点**：中止是"自动"（能而不欲），未遂是"意志以外"（欲而不能），预备是"尚未着手+意志以外"。

##### 犯罪既遂四种类型

| 类型 | 标准 | 立法例 |
|------|------|--------|
| **结果犯** | 发生法定犯罪结果 | 故意杀人罪（第232条） |
| **行为犯** | 法定犯罪行为完成 | 投敌叛变罪（第108条） |
| **危险犯** | 造成法定危险状态 | 放火罪（第114条） |
| **举动犯** | 一经着手即既遂 | 煽动分裂国家罪（第103条） |

### 🧠 记忆诀窍

**紧急避险六条件口诀："起时对目限限"**
- **起**因条件（合法权益受危险）
- **时**间条件（危险正在发生）
- **对**象条件（第三方合法权益）
- **目**的条件（正当避险意图）
- **限**度条件（损害＜所避免损害）
- **限**制条件（不得已 + 职务上特殊限制）

**停止形态四兄弟对比口诀："预备没着手，未遂干不成，中止自己停，既遂全完成"**

**法定处罚梯度记忆**（从严到宽）：
```
既遂 → 分则》未遂 → 可从轻/减轻》预备 → 可从轻/减轻/免除》中止 → 应当免除/减轻
```

**一句话总结**：紧急避险 = 两害相权取其轻（损害＜避免，不得已）；停止形态 = 看阶段（预备/实行）+ 看原因（意志以内→中止，意志以外→未遂/预备）。

### 🔑 核心知识点精要

#### 一、归纳推理概述

**归纳推理**：从特殊到一般的推理——由个别性前提推出一般性结论。

**与演绎推理的对比**：

| 维度 | 演绎推理 | 归纳推理 |
|------|---------|---------|
| 思维方向 | 一般 → 特殊 | 特殊 → 一般 |
| 结论范围 | 不超出前提 | **超出**前提范围 |
| 前提与结论联系 | **必然性**（前提真则结论必真） | **或然性**（前提真结论不一定真） |
| 有效性标准 | 有效/无效（形式判定） | 强度高/低（归纳强度） |

#### 二、完全归纳推理

- **定义**：考察了一类事物的**全部**个别对象，得出关于该类事物的一般性结论
- **公式**：S₁是P，S₂是P，……，Sₙ是P（S₁…Sₙ是S类的全部对象）→ 所有S是P
- **性质**：结论具有**必然性**（实质上是穷举验证，不是真正的归纳推理）
- **局限**：要求穷尽全部对象，实践中适用范围有限

#### 三、不完全归纳推理

##### （一）简单枚举归纳推理

- **定义**：根据某类事物的**部分对象**具有（或不具有）某种属性，且**没有遇到反例**，推出该类全部对象都具有（或不具有）该属性
- **公式**：S₁是P，S₂是P，……，Sₙ是P（S₁…Sₙ是S类的部分对象，未遇反例）→ 所有S是P
- **性质**：结论是**或然**的
- **提高可靠性**：增加样本数量、扩大考察范围、注意是否有反例

##### （二）科学归纳推理

- **定义**：不仅根据部分对象具有某属性，而且分析对象与属性之间的**因果关系或内在必然联系**，从而推出一般性结论
- **与简单枚举的区别**：
  - 简单枚举：以**经验事实**为根据（观察-归纳）
  - 科学归纳：以**因果联系**为根据（观察-分析-归纳）
- **可靠性**：高于简单枚举（因为有因果机理支撑）

#### 四、探求因果联系的逻辑方法——穆勒五法 ⭐⭐

##### 1. 契合法（求同法）

- **基本思路**：考察被研究现象出现的**若干场合**，各场合中**只有一种情况是共同的**，则该共同情况就是被研究现象的原因
- **公式**：
  ```
  场合(1): A、B、C → a
  场合(2): A、D、E → a
  场合(3): A、F、G → a
  ∴ A 是 a 的原因
  ```
- **核心**：**异中求同**——不同场合中找唯一共同先行情况

##### 2. 差异法（求异法）

- **基本思路**：比较被研究现象**出现**的场合与**不出现**的场合，两种场合中**只有一种情况不同**，则该不同情况就是被研究现象的原因
- **公式**：
  ```
  正面场合: A、B、C → a
  反面场合: —、B、C → —（无a）
  ∴ A 是 a 的原因
  ```
- **核心**：**同中求异**——相同背景中找唯一不同先行情况
- **比契合法更可靠**（有对照组）

##### 3. 契合差异并用法

- **基本思路**：将契合法和差异法结合使用
  - 第一步：正事例组 → 用契合法找出共同情况A
  - 第二步：负事例组 → 用契合法找出共同情况（都缺少A）
  - 第三步：对比正负两组 → 用差异法确认A是原因
- **公式**：
  ```
  正事例组: A、B、C → a;  A、D、E → a;  A、F、G → a
  负事例组: —、B、M → —; —、D、N → —; —、P、Q → —
  ∴ A 是 a 的原因
  ```

##### 4. 共变法

- **基本思路**：在其他条件不变的情况下，某一现象**发生变化**，另一现象也随之**发生相应变化**，则前一现象是后一现象的原因
- **公式**：
  ```
  A₁、B、C → a₁
  A₂、B、C → a₂
  A₃、B、C → a₃
  ∴ A 是 a 的原因
  ```
- **核心**：**量变联动**——同方向变化（同增同减）或反向变化

##### 5. 剩余法

- **基本思路**：已知某一复合现象是由另一复合原因引起的，把其中已确定的因果联系部分**减去**，则**剩余**部分也有因果联系
- **公式**：
  ```
  已知：A、B、C、D → a、b、c、d
  已知：A 是 a 的原因，B 是 b 的原因，C 是 c 的原因
  ∴ D 是 d 的原因
  ```
- **核心**：**排除法**——减去已知因果，剩余部分即因果关系

---

### 🧠 记忆诀窍

**穆勒五法口诀："求同求异并用变剩余"**
- **求同**法 = 契合法（异中求同）
- **求异**法 = 差异法（同中求异，有对照组最可靠）
- **并用** = 契合法+差异法组合
- **共变**法（量变联动）
- **剩余**法（排除法）

**五法一句话区别**：

| 方法 | 一句话识别 |
|------|-----------|
| 契合法 | "不同场合，唯此相同" |
| 差异法 | "有此则然，无此则不然" |
| 并用法 | "正向都然+负向都不然" |
| 共变法 | "此变彼亦变" |
| 剩余法 | "已知的都排除，剩下的就是" |

**归纳 vs 演绎速记**：
- 演绎：一般→特殊，**必然**，前提蕴含结论
- 归纳：特殊→一般，**或然**，结论超出前提
