#### 第八章 非营利法人

##### 第一节 非营利法人的概念和分类

**一、非营利法人的概念**

> **【课本原文·法条】** 《民法总则》第87条第1款：为公益目的或者其他非营利目的成立，不向出资人、设立人或者会员分配所取得利润的法人，为非营利法人。

**【课本原文】非营利法人的四项特征：**

| 特征 | 内容 | 关键对比（vs 营利法人） |
|------|------|----------------------|
| ① 不分配利润 | 不向出资人、设立人或者**会员**分配利润 | 营利法人可向股东分红；区别不在是否经营，在**利润是否分配** |
| ② 公益目的 | 基于公益目的设立 | 营利法人基于私益目的（为设立人/成员利益） |
| ③ 组织机构特殊 | 除社会团体法人外，**不设成员大会** | 营利法人应当设权力机构（成员大会） |
| ④ 终止后剩余财产特殊 | 不得向出资人/设立人/会员分配剩余财产（《民法总则》第95条） | 营利法人终止后将财产分配给成员 |

**二、非营利法人的类型**

> **【课本原文·法条】** 《民法总则》第87条第2款：非营利法人包括事业单位、社会团体、基金会、社会服务机构等。

```
                          非营利法人
                              │
          ┌────────┬──────────┼──────────┬────────┐
          │        │          │          │        │
      事业单位  社会团体    捐助法人   宗教活动   其他
      法人      法人       (基金会、  场所法人
                         社会服务
                         机构等)
```

**🧠 课本立场**：营利法人和非营利法人的分类是**我国《民法总则》基于自身国情的独创**，突破了传统大陆法系社团法人和财团法人的界分理论。

**三、非营利法人的终止——剩余财产分配三规则**

> **【课本原文·法条】** 《民法总则》第95条：为公益目的成立的非营利法人终止时，不得向出资人、设立人或者会员分配剩余财产。剩余财产应当按照法人章程的规定或者权力机构的决议用于公益目的；无法按照法人章程的规定或者权力机构的决议处理的，由主管机关主持转给宗旨相同或者相近的法人，并向社会公告。

| 步骤 | 规则 |
|------|------|
| 第一步 | ❌ 不得向出资人/设立人/会员分配 |
| 第二步 | ✅ 按章程或权力机构决议用于公益目的 |
| 第三步 | 🏛️ 无法处理时→主管机关主持转给宗旨相同或相近的法人+公告 |

##### 第二节 事业单位法人

> **【课本原文】** 事业单位法人是相对于企业法人而言的，从事非营利性的社会各项公益事业的各类法人，例如从事新闻、出版、广播、电视、电影、教育、文艺、科研、医疗等事业的法人。

**【课本原文】三特征：**

| 特征 | 内容 |
|------|------|
| ① 非营利 | 不以营利为目的；即使营利活动所得利益只能用于增加资产规模，**不能给股东或成员分红** |
| ② 大多为财团法人 | 以国家财政拨款为基础，无成员，无法设立成员大会。但**本书认为与典型财团法人不同**——不少事业单位没有捐助章程 |
| ③ 靠财政拨款 | 国家财政拨款+独立支配；但**农村集体和个人也可出资举办** |

**设立两种方式（《民法总则》第88条）：**

| 类型 | 法人资格取得时间 |
|------|-----------------|
| 需登记的事业单位 | 经依法登记成立 |
| 不需登记的事业单位 | **从成立之日起**（如银监会/证监会/保监会等参照公务员管理的） |

**组织机构（《民法总则》第89条）：**
- 理事会 = 决策机构（事业单位无成员大会！）
- 法定代表人依法律/行政法规/法人章程产生

---

##### 第三节 社会团体法人

> **【课本原文】** 社会团体法人是指具备法人条件，基于会员共同意愿，为公益目的或者会员共同利益等非营利目的而依法设立的社会团体。

**【课本原文】三特征：**

| 特征 | 内容 |
|------|------|
| ① 基于会员共同意愿设立 | 自愿成立，协商一致 |
| ② 非营利性 | 可为公益或会员共同利益；可从事营利活动但**不得将收入分配给会员** |
| ③ 有独立财产 | 源于成员出资/会费/社会捐助；**设立人和成员不对法人债务负责**（有限责任） |

**组织机构三重结构：**

```
          社会团体法人组织机构
                 │
    ┌────────────┼────────────┐
    │            │            │
 权力机构      执行机构      法定代表人
(会员大会/     (理事会等)    (理事长/会长
 会员代表                   按章程担任)
 大会)
```

> **【课本原文·法条】** 《民法总则》第91条第2款：社会团体法人应当设会员大会或者会员代表大会等权力机构。第3款：社会团体法人应当设理事会等执行机构。

##### 第四节 捐助法人和宗教活动场所法人

> **【课本原文】** 捐助法人，是指具备法人条件，为公益目的以捐助财产设立的基金会、社会服务机构等组织。在比较法上，捐助法人通常被称为私法上的**财团法人**。

**【课本原文】捐助法人四特征：**

```
                   捐助法人的四个特征
                         │
    ┌────────┬───────────┼───────────┬────────┐
    │        │           │           │        │
  非营利性  财产集合体   没有成员     必须登记设立
  (公益目的  (财团法人   (无意思机关   (所有捐助法人
  +不分配)   以财产为    无权力机构)   都必须登记)
            基础)
```

**【课本原文】捐助人三项权利（《民法总则》第94条）：**

| 权利 | 内容 |
|------|------|
| ① 查询权 | 向捐助法人查询捐助财产的使用、管理情况 |
| ② 建议权 | 提出意见和建议 |
| ③ 撤销权 | 决策机构/执行机构/法定代表人决定违反法律或章程→请求法院撤销（但**善意相对人形成的民事法律关系不受影响**） |

**捐助法人 vs 社会团体法人：**

| 维度 | 捐助法人 | 社会团体法人 |
|------|---------|-------------|
| 设立基础 | **财产**的结合（财团法人） | **人**的联合（社团法人） |
| 有无成员 | 无 | 有会员 |
| 设立人与组织关系 | 设立后即脱离，不当然成为管理者 | 设立人一般成为成员 |

**宗教活动场所法人三特点：**
1. 以**财产**为基础（土地/建筑/庙产等财产综合体）→ 财团法人
2. 属于**非营利法人**（可经营但收益只能用于宗教活动）
3. 以从事**宗教活动**为设立目的

> 🏛️ 宗教活动场所 vs 宗教团体：宗教团体属于**社会团体**（如佛教协会），宗教场所属于**捐助法人**。二者彼此独立、互不隶属。

#### 第九章 特别法人

##### 第一节 特别法人概述

> **【课本原文】** 特别法人，是我国法上营利法人和非营利法人之外的一类法人。根据《民法总则》第96条，包括：机关法人、农村集体经济组织法人、城镇农村的合作经济组织法人、基层群众性自治组织法人。特别法人制度为我国《民法总则》**所独创**。

**【课本原文】特别法人三特征：**

| 特征 | 说明 |
|------|------|
| ① 营利/非营利无法涵盖 | 特别法人类别是采营利/非营利二分法的**必然结果**（机关法人无法归入任一） |
| ② 设立特殊 | 依据法律/行政命令设立，非基于出资人意愿 |
| ③ 权利能力和行为能力特殊 | 只能从事"为履行职能所需要的民事活动"（《民法总则》第97条、第101条） |

**【课本原文·本书批评】** 采用封闭式列举（四种，无"等"字）未必十分妥当：
- 民办学校/民办医院/民办养老院等**兼有公益性与营利性**的组织，法律地位很难界定
- 随经济社会发展，新类型法人应可纳入→**特别法人应保持开放性**

---

##### 第二节 机关法人

> **【课本原文】** 机关法人，是指依照法律和行政命令组建的，享有公权力的以从事国家管理活动为主的各级国家机关。

**【课本原文】五特征：**

| # | 特征 | 说明 |
|---|------|------|
| ① | 设立特殊 | 依法律法规和行政命令成立，**非基于出资人意愿** |
| ② | 不需登记 | 成立之日起即具有法人资格 |
| ③ | 享有公权力 | 代表国家从事管理；只有从事**民事活动**时才以机关法人身份出现 |
| ④ | 财产来自财政拨款 | 有独立经费（非独立财产），通过预算拨款取得 |
| ⑤ | 行为能力严格受限 | **不能从事经营活动**；不动产只享有占有/使用/依法处分，**不享有收益权**（《物权法》第53条） |

**设立三条件（《民法总则》第97条）：**
1. 有独立经费
2. 承担行政职能
3. 经特许成立（采**特许主义**，不同于营利/非营利法人的准则主义/许可主义）

**终止规则（《民法总则》第98条）：**
- 被撤销→法人终止
- 权利义务由**继任机关法人**享有和承担
- 无继任的→由**作出撤销决定的机关法人**享有和承担
- ❌ 不适用破产程序（机关法人不能破产）

##### 第三节 农村集体经济组织法人和合作经济组织法人

**农村集体经济组织法人四特征：**

| 特征 | 内容 |
|------|------|
| ① 财产特殊性 | 主要是**集体所有**的财产（土地等），源于《宪法》第10条第2款 |
| ② 成员身份性 | 成员加入/退出与身份密切相关，**不是自由进入/自由退出**（vs 营利法人） |
| ③ 职能特定性 | 经营/管理农民集体所有土地，从事农业生产经营（广义农业） |
| ④ 组织体稳定性 | 不能随意解散/清算，非经法律规定不能消灭 |

**合作经济组织法人五特征：**

| # | 特征 | 说明 |
|---|------|------|
| ① | 劳动者自愿联合 | 自己出资/自己经营/自己劳动，**不接受非社员投资** |
| ② | 不以营利为主要目的 | 社员间互济/互助/互惠/互利，为社员服务 |
| ③ | 非按资分配 | 按交易额分配/按劳分配 |
| ④ | 劳动民主 | 社员共同经营，**均享有平等投票权**（vs 公司股份民主） |
| ⑤ | 社员参与经营/劳动 | 劳动质量/数量是收益分配重要依据 |

> 🧠 **课本立场**：合作经济组织法人性质存在争议（营利？非营利？中间？），立法机关认为"既无法归入营利也无法归入非营利"→**放入特别法人**。

##### 第四节 基层群众性自治组织法人

**【课本原文】两会四特殊性：**

| 特殊性 | 居委会 | 村委会 |
|--------|--------|--------|
| ① 设立 | 依据法律直接设立，不需登记 | 同上 |
| ② 职能 | 法定公益+公共职能（六项任务） | 自我管理/教育/服务，民主选举/决策/管理/监督 |
| ③ 组织机构 | 成员经居民直接选举 | 主任+副主任+委员共**3-7人**，应有妇女成员 |
| ④ 财产 | 经费由政府划拨 | 经费困难时政府提供，公益事业由村民会议筹资筹劳 |

> **【课本原文·法条】** 《民法总则》第101条第2款：未设立村集体经济组织的，村民委员会可以依法代行村集体经济组织的职能。

#### 第十章 非法人组织

##### 第一节 非法人组织概述

> **【课本原文·法条】** 《民法总则》第102条第1款：非法人组织，是指不具有法人资格，但是能够依法以自己的名义从事民事活动的组织。

**【课本原文】四特征：**

```
                   非法人组织的四个特征
                         │
    ┌────────┬───────────┼───────────┬────────┐
    │        │           │           │        │
  依法设立   能以自己的   也可能有     不能独立
  (原则上    名义从事     自己的财产   承担责任 ⭐
  应登记)   民事活动     (先以该财产   (出资人/设立人
            (独立民事    清偿)        承担无限责任)
            主体)
```

> 🔴 **核心区分**：法人→独立责任（成员有限责任）vs 非法人组织→**不独立责任（出资人/设立人无限责任）**

**非法人组织 vs 法人（三区别）：**

| 维度 | 法人 | 非法人组织 |
|------|------|-----------|
| 独立财产 | 有独立财产，与成员财产分离 | 无独立财产，财产由成员**共有** |
| 独立责任 | 独立承担责任，成员**有限责任** | 不独立承担责任，出资人/设立人**无限责任** |
| 成员与团体关系 | 成员不一定直接参与经营 | 成员一般**直接参与**经营管理 |

**三种类型：**

| 类型 | 出资人数 | 责任形式 | 法律依据 |
|------|---------|---------|---------|
| 个人独资企业 | 1个自然人 | 无限责任 | 《个人独资企业法》 |
| 合伙企业 | 2个以上 | 普通合伙人无限连带/有限合伙人以出资为限 | 《合伙企业法》 |
| 不具有法人资格的专业服务机构 | 视具体类型 | 特殊普通合伙等形式 | 《律师法》《注册会计师法》 |

---

##### 第二~四节 设立·财产责任·解散

**【课本原文】财产责任（《民法总则》第104条）——两个层次：**

```
     债务清偿顺序
          │
          ▼
  ┌──────────────────┐
  │ 第一层：以非法人   │  ← 相对独立责任
  │ 组织自己的财产先偿 │
  └──────┬───────────┘
         │ 不足时
         ▼
  ┌──────────────────┐
  │ 第二层：出资人/   │  ← 无限连带责任
  │ 设立人承担无限责任 │     （法定，不得约定排除）
  └──────────────────┘
```

> 📌 **例外**：有限合伙人以认缴出资额为限（《合伙企业法》第2条）；个人独资企业解散后债权人在**5年内**未提出偿债请求→责任消灭（《个人独资企业法》第28条）

**【课本原文】解散三事由（《民法总则》第106条）：**
1. 章程规定的存续期间届满或解散事由出现
2. 出资人或设立人决定解散（**私法自治**，任何时候都可决定）
3. 法律规定的其他情形（强制解散）

### 🧠 记忆诀窍

#### 口诀

**非营利法人终止规则**：「不分→章用→官转」
- **不分**：不得向出资人/设立人/会员分配
- **章用**：按章程/决议用于公益
- **官转**：主管机关主持转给宗旨相同/相近法人

**捐助人三权利**：「查明撤」
- **查**：查询财产使用管理
- **明**：提出意见和建**议**（谐音）
- **撤**：请求法院撤销违法决定

**特别法人四种**：「机农合居」
- 机关法人、农村集体经济组织、合作经济组织、基层群众自治组织

**非法人组织四特征**：「设名财责」
- **设**：依法设立（登记）
- **名**：以自己**名**义活动
- **财**：可能有自己财产
- **责**：不独立承担**责**任（无限！）

#### 一句话速记

| 概念 | 一句话 |
|------|--------|
| 非营利法人 | 不为分钱为公益，不分利润是核心 |
| 事业单位法人 | 财团为主靠拨款，理事会说了算 |
| 社会团体法人 | 人的联合有会员，大会+理事会二重奏 |
| 捐助法人 | 财产集合无成员，捐助人可查可撤 |
| 特别法人 | 营利非营利都装不下，法律特设第四类 |
| 机关法人 | 有经费有公权，成立即法人，不能去经营 |
| 非法人组织 | 能活动不能独立扛，出了事无限责任兜底 |
| 非法人vs法人 | 财产不独立→责任不独立，这是根本 |

| 对比项 | A | B | 区分点 |
|--------|---|---|--------|

#### 第七章 犯罪主体

##### 第一节 犯罪主体概述

> **【课本原文】** 犯罪主体是指实施危害社会的行为并依法应当负刑事责任的自然人和单位。自然人可成为所有犯罪的主体，单位只可成为部分犯罪的主体。

##### 第二节 自然人犯罪主体

**一、自然人犯罪主体三条件**

| 条件 | 内容 |
|------|------|
| ① 达刑事责任年龄 | 《刑法》第17条的分龄规定 |
| ② 具有刑事责任能力 | 辨认能力+控制能力 |
| ③ 实施了刑法规定的危害行为 | 罪刑法定原则要求 |

**二、刑事责任年龄 ⭐⭐⭐（《刑法》第17条——四档+一从宽）**

```
         《刑法》第17条 刑事责任年龄体系
                    │
    ┌───────────────┼───────────────┐
    │               │               │
  完全不负         相对负            完全负
  刑事责任         刑事责任          刑事责任
                    │
          ┌─────────┴─────────┐
          │                   │
    特殊相对负            相对负
    (12-14周岁)          (14-16周岁)
```

| 年龄档 | 范围 | 责任范围 | 法条依据 |
|--------|------|---------|---------|
| 完全不负 | 不满12周岁 | 不负任何刑事责任 | — |
| **特殊相对负** | 12-14周岁 | 犯故意杀人、故意伤害罪→致人死亡或以特别残忍手段致人重伤造成严重残疾+情节恶劣→**经最高检核准追诉** | 《刑法》第17条第3款 |
| **相对负** | 14-16周岁 | 8种犯罪：故意杀人、故意伤害致人重伤或死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投放危险物质 | 《刑法》第17条第2款 |
| **完全负** | 已满16周岁 | 刑法规定的**所有**犯罪 | 《刑法》第17条第1款 |

> ⚠️ **未成年人从宽**：已满12周岁不满18周岁的人犯罪，**应当**从轻或者减轻处罚（《刑法》第17条第4款）

**📌 重要争议（课本立场）**：
- 14-16周岁"8种犯罪"是指**具体犯罪行为**而非具体罪名（全国人大法工委2002年答复）→ 绑架撕票的也应追究
- 14-16周岁的"抢劫"是否包括转化型抢劫（第269条）和携带凶器抢夺（第267条第2款）？**本书认为从刑法谦抑性考虑应排除**

**三、刑事责任能力**

> **【课本原文】** 刑事责任能力的本质是人实施行为时具备相对自由的意志能力，即行为人实施刑法所禁止的严重危害社会的行为，具备相对自由的认识和抉择行为的能力。

**【课本原文】辨认行为能力 vs 控制行为能力：**

| 维度 | 辨认行为能力 | 控制行为能力 |
|------|------------|------------|
| 含义 | 对行为在刑法上的意义/性质/后果的分辨认识能力 | 决定自己是否以行为触犯刑法的能力 |
| 关系 | **基础**——先有认识才能谈控制 | **关键**——最终体现责任能力 |

> 🧠 二者**必须同时具备**，才可认定为具有刑事责任能力。

**【课本原文】四分法：**

| 类型 | 情形 | 责任 |
|------|------|------|
| 完全刑事责任能力 | 正常成年人 | 负完全责任 |
| 完全无刑事责任能力 | 精神病人不能辨认/控制自己行为 | 不负责任 |
| **减轻（限定）刑事责任能力** | ①12-18周岁未成年人 ②又聋又哑的人 ③盲人 ④尚未完全丧失辨认/控制能力的精神病人 | 可从宽 |
| 相对无刑事责任能力 | （特定年龄段的特殊处理） | — |

**【课本原文】生理性醉酒 vs 病理性醉酒：**

| 类型 | 责任 | 说明 |
|------|------|------|
| 生理性醉酒 | 一般应负完全刑事责任（《刑法》第18条第4款） | 如果醉酒前对犯罪有故意/过失→完全责任；否则应依实际精神状态确定 |
| 病理性醉酒 | 区分情形 | 故意/过失使自己陷于病理性醉酒→仍可能负完全责任 |

**四、犯罪主体的特殊身份**

| 分类 | 含义 | 举例 |
|------|------|------|
| **定罪身份**（真正身份犯） | 决定刑事责任**有无**的身份 | 贪污罪需国家工作人员 |
| **量刑身份**（不真正身份犯） | 影响刑事责任**大小**的身份 | 国家机关工作人员犯诬告陷害罪从重处罚 |

> 📌 特殊身份一般是**行为开始时**就已具有的，行为后才形成的地位（如聚众犯罪的首要分子）不属于特殊身份。

##### 第三节 单位犯罪主体

> **【课本原文·法条】** 《刑法》第30条：公司、企业、事业单位、机关、团体实施的危害社会的行为，法律规定为单位犯罪的，应当负刑事责任。

**【课本原文】单位犯罪特征：**
- 以单位名义实施
- 为单位谋取非法利益
- 体现单位意志（经单位决策程序）
- **法律规定为单位犯罪的才追究刑事责任**（法定性！）

| 处罚原则 | 内容 |
|---------|------|
| **双罚制**（原则） | 对单位判处罚金 + 追究直接责任人员刑事责任 |
| **单罚制**（例外） | 法律另有规定时仅处罚直接责任人员 |

#### 第八章 犯罪主观方面

##### 第一节 概述

> **【课本原文】** 犯罪主观方面，是指行为人对其实施的行为所必然或可能引起的危害社会的结果所持的心理态度。

**【课本原文】三因素：**
- **罪过**（故意+过失）——必备要素，核心内容
- **犯罪目的**——选择要素（仅存在于某些故意犯罪）
- **犯罪动机**——非独立构成要素，但对定罪量刑有影响

**四特征：**
1. 支配行为的**心理状态**
2. 是行为人承担刑事责任的**主观基础**
3. 是**刑法明文规定**的心理状态（法定性）
4. 是**一切犯罪必须具备**的要件（主客观相统一原则）

---

##### 第二节 犯罪故意 ⭐⭐⭐

> **【课本原文·法条】** 《刑法》第14条第1款：明知自己的行为会发生危害社会的结果，并且希望或者放任这种结果发生，因而构成犯罪的，是故意犯罪。

**【课本原文】犯罪故意 = 认识因素 + 意志因素：**

```
               犯罪故意
                  │
      ┌───────────┴───────────┐
      │                       │
   认识因素                  意志因素
  （明知）                  （希望/放任）
  "明知自己的行为            "希望或者放任
  会发生危害社会             这种结果发生"
  的结果"
      │                       │
  前提条件                  决定性因素
```

**直接故意 vs 间接故意对比 ⭐：**

| 维度 | 直接故意 | 间接故意 |
|------|---------|---------|
| 认识因素 | 明知**必然或可能**发生 | 明知**可能**发生 |
| 意志因素 | **希望**结果发生 | **放任**结果发生（听之任之、满不在乎） |
| 未发生结果 | 成立未遂 | 不成立犯罪 |

**【课本原文】间接故意三种典型情形：**
1. 追求非犯罪目的时，放任另一危害结果（如防盗私设电网致人伤亡）
2. 追求犯罪目的时，放任另一危害结果（如盗窃井盖放任车辆损伤）
3. 突发性犯罪中，临时起意，不计后果（如被抓后为挣脱而捅刀）

**犯意转化 vs 另起犯意：**

| 维度 | 犯意转化 | 另起犯意 |
|------|---------|---------|
| 含义 | **同一犯罪过程**中故意内容变化 | 停止原犯罪意图后**产生新的**犯罪意图 |
| 侵害法益 | 同一或同类法益 | 多数情况下不相同 |
| 处理 | 按一罪处理 | 按数罪并罚 |

---

##### 第三节 犯罪目的与犯罪动机

| 维度 | 犯罪目的 | 犯罪动机 |
|------|---------|---------|
| 含义 | 希望通过实施犯罪行为**达到**的某种危害结果 | **刺激**犯罪人实施犯罪行为的内心起因 |
| 作用 | 决定犯罪行为的方向（"做什么"） | 推动犯罪行为的动力（"为什么做"） |
| 时间顺序 | 形成于后 | 产生于前 |
| 定罪意义 | **目的犯**以特定目的为构成要件 | 一般不直接决定定罪 |

---

##### 第四节 犯罪过失 ⭐⭐⭐

> **【课本原文·法条】** 《刑法》第15条第1款：应当预见自己的行为可能发生危害社会的结果，因为疏忽大意而没有预见，或者已经预见而轻信能够避免，以致发生这种结果的，是过失犯罪。

**疏忽大意的过失 vs 过于自信的过失：**

| 维度 | 疏忽大意的过失（无认识过失） | 过于自信的过失（有认识过失） |
|------|---------------------------|---------------------------|
| 认识状态 | **应当预见**但**没有预见** | **已经预见**但**轻信能避免** |
| 意志态度 | 疏忽大意 | 过于自信 |
| 公式 | 应预见+未预见 | 已预见+轻信避免 |

**过于自信的过失 vs 间接故意（必考对比！）：**

| 维度 | 过于自信的过失 | 间接故意 |
|------|--------------|---------|
| 相同点 | 都预见到危害结果**可能**发生 | 都预见到危害结果**可能**发生 |
| 对结果的态度 | **不希望**发生，轻信能避免 | **放任**发生（发生与否都不违背本意） |
| 客观依据 | 有**避免结果**的客观依据（如技术/经验） | 无任何避免措施 |

---

##### 第五节 无罪过行为

| 类型 | 核心 | 法条 | 责任 |
|------|------|------|------|
| **意外事件** | 行为客观上造成损害，主观上**不能预见** | 《刑法》第16条 | 不负刑事责任 |
| **不可抗力** | **不能抗拒**的原因造成损害 | 《刑法》第16条 | 不负刑事责任 |
| **无期待可能性** | 法律不强人所难，不能期待行为人做出合法行为 | — | 极为罕见，仅极端情形 |

##### 第六节 刑法上的认识错误

```
              认识错误
                 │
    ┌────────────┴────────────┐
    │                         │
 法律认识错误              事实认识错误
 (对法律性质认识错误)       (对构成要件事实认识错误)
    │                         │
 原则上不影响定罪             │
 (不知法律不免责)      ┌──────┴──────┐
                       │             │
                  具体事实错误   抽象事实错误
                  (同一构成要件)  (不同构成要件)
                      │
              ┌───┬───┼───┬───┐
              │   │   │   │   │
            对象  手段  因果  打击
            错误  错误  关系  错误
                        错误
```

| 类型 | 处理原则 |
|------|---------|
| 法律认识错误 | 原则上**不影响定罪**；例外：因客观原因无法知晓法律时可能减轻/免除 |
| 事实认识错误 | 根据**主客观相统一**原则具体分析 |

### 🧠 记忆诀窍

#### 口诀

**刑事责任年龄——「一二三四档，十七来从宽」**
- 一（12-14）：特殊相对负→故意杀人/伤害+严重+最高检核准
- 二（14-16）：相对负→8种罪
- 三（16+）：完全负→全部犯罪
- 四（-12）：完全不负
- 十七条第4款：12-18周岁**应当**从轻/减轻

**8种犯罪口诀**：「杀杀强强抢，毒火爆投」
- 故意杀人、故意伤害致人重伤/死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投放危险物质

**直接故意 vs 间接故意**：
- 直接：「明知必然/可能，希望发生」→ 不发生时是未遂
- 间接：「明知可能，放任发生」→ 不发生时无犯罪

**过于自信 vs 间接故意**：
- 过于自信：「我不希望，我能避免」（轻信）
- 间接故意：「发不发生在乎，无所谓」（放任）

**疏忽大意 vs 过于自信**：
- 疏忽：「该想到的没想到」
- 过于自信：「想到了但觉得不会」

#### 一句话速记

| 概念 | 一句话 |
|------|--------|
| 刑事责任能力 | 辨认是基础，控制是关键，二者必须同时有 |
| 单位犯罪 | 有法定才追究，双罚是原则 |
| 犯罪故意 | 知道+想要/放任 |
| 间接故意 | 听之任之，发不发生都无所谓 |
| 犯意转化 | 同一过程变了心→一罪 |
| 另起犯意 | 停了旧的起新的→数罪 |
| 意外事件 | 不能预见→不担责 |
| 法律认识错误 | 不知法不免责（原则） |

#### 关键对比速记表

| 对比项 | A | B | 区分标志 |
|--------|---|---|---------|
| 直接故意 vs 间接故意 | 认识+希望 | 认识+放任 | 意志因素不同 |
| 间接故意 vs 过于自信 | 放任结果 | 轻信避免 | 对结果的态度 |
| 疏忽大意 vs 过于自信 | 未预见 | 已预见 | 认识状态 |
| 犯意转化 vs 另起犯意 | 一罪 | 数罪 | 是否同一过程 |

### 🔑 核心知识点精要

#### 一、联言推理

| 类型 | 公式 | 说明 |
|------|------|------|
| **分解式** | (p∧q) → p | 由联言命题真→任一支命题真 |
| **组合式** | p, q → (p∧q) | 由各支命题真→联言命题真 |

---

#### 二、选言推理 ⭐

```
               选言推理
                  │
    ┌─────────────┴─────────────┐
    │                           │
  相容选言推理              不相容选言推理
  (p∨q)                     (p⊻q)
    │                           │
 否定肯定式 ✅              否定肯定式 ✅
 (p∨q)∧¬p → q             (p⊻q)∧¬p → q
    │                           │
 肯定否定式 ❌              肯定否定式 ✅
 (不能使用！)               (p⊻q)∧p → ¬q
```

> 🔴 **关键记忆**：相容选言推理**只有否定肯定式有效**，肯定否定式无效！（因为相容意味着可以同时为真）

---

#### 三、假言推理 ⭐⭐

**充分条件假言推理（p→q）：**

| 有效式 | 公式 | 无效式 | 公式 |
|--------|------|--------|------|
| **肯定前件式** ✅ | (p→q)∧p → q | 否定前件式 ❌ | (p→q)∧¬p → ¬q |
| **否定后件式** ✅ | (p→q)∧¬q → ¬p | 肯定后件式 ❌ | (p→q)∧q → p |

> 🧠 记忆：「充分→背前背后都错」（否定前件和肯定后件都无效）

**必要条件假言推理（p←q）：**

| 有效式 | 公式 | 无效式 | 公式 |
|--------|------|--------|------|
| **否定前件式** ✅ | (p←q)∧¬p → ¬q | 肯定前件式 ❌ | (p←q)∧p → q |
| **肯定后件式** ✅ | (p←q)∧q → p | 否定后件式 ❌ | (p←q)∧¬q → ¬p |

> 🧠 记忆：「必要→背前肯后都错」（肯定前件和否定后件都无效）

**充分条件 vs 必要条件的有效式对比：**

```
  充分条件(p→q)          必要条件(p←q)
  ┌─────────────┐       ┌─────────────┐
  │ 肯定前件 ✅  │       │ 肯定前件 ❌  │
  │ 否定后件 ✅  │       │ 否定后件 ❌  │
  │ 否定前件 ❌  │       │ 否定前件 ✅  │
  │ 肯定后件 ❌  │       │ 肯定后件 ✅  │
  └─────────────┘       └─────────────┘
  
  规律：充分和必要的有效式恰好相反！
```

---

#### 四、二难推理（假言选言推理）

**简单构成式：**
```
如果p则r，如果q则r
p或q
∴ r
```
公式：((p→r)∧(q→r)∧(p∨q)) → r

**简单破坏式：**
```
如果p则q，如果p则r
非q或非r
∴ 非p
```
公式：((p→q)∧(p→r)∧(¬q∨¬r)) → ¬p

**复杂构成式（最常见）：**
```
如果p则r，如果q则s
p或q
∴ r或s
```

---

### 🧠 记忆诀窍

**充分条件假言推理——「肯前否后」有效**
- **肯**定**前**件 ✅
- **否**定**后**件 ✅
- 其他无效

**必要条件假言推理——「否前肯后」有效**
- **否**定**前**件 ✅
- **肯**定**后**件 ✅
- 其他无效

**一句话记规律**：充分和必要有效式**方向相反**——充分→顺着箭头（肯前→肯后，否后→否前），必要→逆着箭头（否前→否后，肯后→肯前）

**相容选言推理**：「只能否肯，不能肯否」（只能否定肯定式，不能肯定否定式）

> 💡 **类比**：相容选言就像自助餐——你知道某人没吃牛肉(p为假)，可以断定他吃了鸡肉(q为真)；但你知道他吃了牛肉(p为真)，不能断定他没吃鸡肉(q为假)，因为他可能牛肉鸡肉都吃了！
