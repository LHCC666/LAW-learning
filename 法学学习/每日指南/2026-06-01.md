### 二、核心知识点精要

**1. 民法的概念**
- **形式民法**：以"民法"命名的法典（《民法典》）
- **实质民法**：所有调整民事关系的法律规范总称（《民法典》+ 民事单行法 + 司法解释）
- 我国《民法典》2020年5月28日通过，2021年1月1日施行，**7编1260条**

**2. 调整对象** ⭐
- **平等主体之间**的人身关系和财产关系
- 人身关系：人格权、身份权
- 财产关系：物权、债权、继承权、知识产权
- ❌ 不包括：行政隶属关系、财政税收关系（纵向不平等关系）

**3. 民法的性质** ⭐⭐⭐（核心考点，三点）
- ① **私法**：调整平等主体关系，区别于调整国家公权力的公法
- ② **权利法**：以确认和保护民事权利为核心（民法典是"权利宣言书"）
- ③ **市场经济基本法**：确认市场主体的法律地位，规范交易行为

**4. 民法典的体系** ⭐
- **潘德克顿体系**（德国式，我国采用）：
  - 总则→分则，提取公因式
  - 总则编 + 物权编 + 合同编 + 人格权编 + 婚姻家庭编 + 继承编 + 侵权责任编
- **法学阶梯体系**（罗马式/法国式）：人→物→诉讼
- ⚡ 我国特色亮点：**人格权独立成编**（全球首创）、侵权责任独立成编

**5. 民法渊源** ⭐
- ① 法律（《民法典》+ 民事单行法）
- ② 行政法规
- ③ 司法解释（最高人民法院）
- ④ 习惯（《民法典》第10条："法律没有规定的，可以适用习惯，但是不得违背公序良俗"）

### 三、记忆诀窍 🧠

| 要点 | 口诀/编码 |
|------|-----------|
| 民法性质三要素 | **"私权市"** — 私法、权利法、市场经济基本法 |
| 调整对象 | **"平等二人"** — 平等主体的两种关系（人身+财产）|
| 民法典7编 | **"总物合人格，婚继侵"** — 总则、物权、合同、人格权、婚姻家庭、继承、侵权责任 |
| 渊源类型 | **"法行司习"** — 法律→行政法规→司法解释→习惯 |

**对比记忆**：
| | 形式民法 | 实质民法 |
|--|---------|---------|
| 形态 | 成文民法典 | 所有民事规范 |
| 例 | 《民法典》 | 《民法典》+《公司法》+司法解释+… |

## ⚖️ 刑法总论（~50min）— 绪论 + 第1章 刑法概说

### 一、今日章节与范围

| 章节 | 内容 | 重要度 |
|------|------|--------|
| 绪论 | 刑法学概述、沿革 | ⭐ |
| 第1章第1节 | 刑法的概念、性质与渊源 | ⭐⭐ |
| 第1章第2节 | 刑法的指导思想、根据与任务 | ⭐ |
| 第1章第3节 | 刑法的体系与解释 ⭐ | ⭐⭐⭐ |

### 二、核心知识点精要

**1. 刑法的概念与性质**
- **定义**：规定犯罪、刑事责任和刑罚的法律规范的总称
- **性质**：
  - ① **公法**：刑法调整国家与个人关系（与民法截然相反！）
  - ② **强行法**：不允许当事人选择适用
  - ③ **保障法/后盾法**：刑法是其他部门法的"最后一道防线"

**2. 刑法渊源** ⭐
- ① **刑法典**（1997刑法 + 12个修正案）
- ② **单行刑法**（如《关于惩治骗购外汇、逃汇和非法买卖外汇犯罪的决定》）
- ③ **附属刑法**（其他法律中的刑事条款）
- ④ **立法解释 + 司法解释**

**3. 刑法的体系**
- 总则（第1-101条）+ 分则（第102-452条）+ 附则
- 法律条文结构：条→款→项

**4. 刑法的解释** ⭐⭐⭐（重中之重！）

| 解释方法 | 定义 | 例子 |
|----------|------|------|
| **文义解释** | 按条文通常字面含义解释 | "故意杀人"→字面理解 |
| **体系解释** | 联系刑法其他条文综合理解 | 总则"故意"→分则各罪"故意"参照总则 |
| **历史解释** | 考察立法背景和沿革 | 查阅修正案的立法理由 |
| **目的解释** | 根据刑法保护的法益目的解释 | 为什么某种行为被规定为犯罪？ |

**5. 罪刑法定原则预热**（明日重点，今天先感知）
- "法无明文规定不为罪，法无明文规定不处罚"
- 禁止类推、禁止事后法、禁止不定期刑、要求明确性

### 三、记忆诀窍 🧠

| 要点 | 口诀/编码 |
|------|-----------|
| 刑法三性质 | **"公强盾"** — 公法、强行法、后盾法（保障法）|
| 解释四方法 | **"文体历目"** — 文义、体系、历史、目的 |
| 刑法渊源 | **"典单附释"** — 刑法典→单行刑法→附属刑法→解释 |

**对比记忆 — 民法 vs 刑法**：
| | 民法 | 刑法 |
|--|------|------|
| 性质 | **私法** | **公法** |
| 调整关系 | 平等主体之间 | 国家与个人之间 |
| 核心功能 | 确权+救济 | 惩罚+预防 |
| 主体地位 | 平等 | 不平等（国家追诉） |

## 🧠 法律逻辑学（~40min）— 第1章 引论 + 第2章 概念

### 一、今日章节与范围

| 章节 | 内容 | 重要度 |
|------|------|--------|
| 第1章 引论 | 逻辑学的性质与功能、司法工作者学逻辑的必要性 | ⭐ |
| 第2章 概念 | 概念的特征、内涵与外延、分类、概念间关系 | ⭐⭐⭐ |

### 二、核心知识点精要

**1. 逻辑学的性质与功能**
- **性质**：研究思维形式结构及其规律的科学
  - 工具性（思维工具）
  - 全人类性（无阶级性）
- **功能**：训练严密思维、识别逻辑错误、正确表达和论证
- 司法场景：分析案件事实→识别法律规范逻辑结构→推理裁判

**2. 概念的特征** ⭐
- **概念**：反映对象特有属性的思维形式
- 概念 vs 语词：
  - 同一概念可用不同语词表达（"配偶" = "丈夫/妻子"）
  - 同一语词可表达不同概念（"拘留"——刑事拘留 vs 行政拘留 vs 司法拘留）
- **法律概念的特殊性**：具有规定性（基于立法者定义，如"善意第三人"）

**3. 内涵与外延** ⭐⭐（核心区分！）
- **内涵**：概念所反映对象的**特有属性**（是什么）
- **外延**：概念所指对象的**范围**（有哪些）
- **反变关系**：内涵越多→外延越小；内涵越少→外延越大
  - 例："法学生" 内涵少、外延大 → "认真复习的法学生" 内涵多、外延小

**4. 概念的分类** ⭐

| 分类标准 | 类型 | 例子 |
|----------|------|------|
| 按外延对象数量 | **单独概念** | 北京、鲁迅、民法典 |
| | **普遍概念** | 城市、作家、法律 |
| 按是否集合体 | **集合概念** | 森林、法院、陪审团 |
| | **非集合概念** | 树、法官 |
| 按有无某属性 | **正概念（肯定）** | 合法、善意、成年 |
| | **负概念（否定）** | 非法、恶意、未成年 |

> ⚠️ 易错点："法院"作为整体是集合概念；"法官"指个体是非集合概念。"我是法院的人"≠"我是法院"——前者用的是集合概念。

**5. 概念间关系** ⭐⭐⭐（欧拉图必考！）

| 关系 | 特征 | 欧拉图 | 例子 |
|------|------|--------|------|
| **全同关系** | 外延完全相同 | 两圆重合 | 等边三角形 = 等角三角形 |
| **种属关系** | A外延全部包含于B | A在B内 | 法官 ⊂ 法律工作者 |
| **属种关系** | A外延全部包含B | B在A内 | 法律工作者 ⊃ 法官 |
| **交叉关系** | 部分外延重合 | 两圆交叉 | 律师 ∩ 党员 |
| **全异关系** | 外延完全不重合 | 两圆分离 | 合法行为 ∥ 非法行为 |
| └ 矛盾关系 | 全异 + 外延之和 = 属概念外延 | 各占一半 | 合法 + 非法 = 行为 |
| └ 反对关系 | 全异 + 外延之和 < 属概念外延 | 有中间地带 | 红色 + 蓝色 < 颜色 |

### 三、记忆诀窍 🧠

**5种关系的欧拉图速记口诀**：**"同种属交异"** — 全同、种属、属种、交叉、全异

| 关键词 | 编码 |
|--------|------|
| 内涵vs外延 | **"内涵=含什么，外延=延到哪里"** |
| 反变关系 | **内涵越大外延越小**（像个漏斗，滤得越细剩下的越少）|
| 集合vs非集合 | 集合：整体不可分；非集合：每个个体都有该属性 |

**概念关系速查卡**：
```
全同 ⊙  种属 ⊂  属种 ⊃  交叉 ∩  全异 ∥
矛盾 = 全异 + 穷尽（合法/非法）
反对 = 全异 + 未穷尽（红/蓝，还有绿黄紫…）
```
